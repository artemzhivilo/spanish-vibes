# Spanish Vibes

A TikTok-style Spanish language learning app with adaptive difficulty, AI-powered conversations, and spaced repetition.

Built with FastAPI, HTMX, Tailwind CSS, and SQLite.

## Run locally

```bash
uv run spanish-vibes
```

Then open http://localhost:8000
