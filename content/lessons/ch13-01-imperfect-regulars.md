# Lesson Text

The **present subjunctive** expresses desires, doubts, and possibilities. Its forms come from the **yo** form of the present indicative. Mastering this pattern lets you plug the subjunctive into complex sentences without hesitation.

## Step 1: Start with the yo form

1. Conjugate the verb in the present indicative **yo** form.
2. Drop the final **-o**.
3. Add subjunctive endings.

### Regular -ar verbs → endings **-e, -es, -e, -emos, -éis, -en**

| Pronoun | Hablar |
| --- | --- |
| yo | hable |
| tú | hables |
| él / ella / usted | hable |
| nosotros / nosotras | hablemos |
| vosotros / vosotras | habléis |
| ellos / ellas / ustedes | hablen |

### Regular -er / -ir verbs → endings **-a, -as, -a, -amos, -áis, -an**

| Pronoun | Comer | Vivir |
| --- | --- | --- |
| yo | coma | viva |
| tú | comas | vivas |
| él / ella / usted | coma | viva |
| nosotros / nosotras | comamos | vivamos |
| vosotros / vosotras | comáis | viváis |
| ellos / ellas / ustedes | coman | vivan |

## Stem-changing verbs keep their changes (with one caveat)

- **-ar / -er** stem-changers (e→ie, o→ue) change in all forms **except** nosotros/vosotros.
  - `pensar` → piense, pienses, piense, **pensemos**, penséis, piensen
  - `volver` → vuelva, vuelvas, vuelva, **volvamos**, volváis, vuelvan
- **-ir** stem-changers change everywhere **and** show a shortened change in nosotros/vosotros.
  - `pedir` (e→i) → pida, pidas, pida, pidamos, pidáis, pidan
  - `dormir` (o→ue) → duerma, duermas, duerma, **durmamos**, durmáis, duerman
  - `preferir` (e→ie) → prefiera, prefieras, prefiera, **prefiramos**, prefiráis, prefieran

## Irregular yo forms carry over

Any irregularity in the yo form appears throughout the subjunctive.

| Yo form | Subjunctive |
| --- | --- |
| tengo → tenga | tenga, tengas, tenga, tengamos, tengáis, tengan |
| hago → haga | haga, hagas... |
| conozco → conozca | conozca, conozcas... |
| digo → diga | diga, digas... |

## Six totally irregular verbs

| Infinitive | Present subjunctive |
| --- | --- |
| ser | sea, seas, sea, seamos, seáis, sean |
| ir | vaya, vayas, vaya, vayamos, vayáis, vayan |
| haber | haya, hayas, haya, hayamos, hayáis, hayan |
| estar | esté, estés, esté, estemos, estéis, estén |
| saber | sepa, sepas, sepa, sepamos, sepáis, sepan |
| dar | dé, des, dé, demos, deis, den |

## Spelling changes (orthographic adjustments)

To preserve pronunciation before `-e/-a` endings:

- `-car` → qu: `buscar` → busque, busques…
- `-gar` → gu: `pagar` → pague, pagues…
- `-zar` → c: `empezar` → empiece, empieces…
- `-ger/-gir` → j: `escoger` → escoja…
- `-guir` → g: `seguir` → siga, sigas…

## Guided Practice

### Build the forms

1. `Nosotros ___ (hablar)` → `hablemos`
2. `Ellos ___ (comer)` → `coman`
3. `¿Qué tú ___ (vivir)?` → `vivas`
4. `Yo ___ (tener)` → `tenga`
5. `Ana ___ (pensar)` → `piense`

### Apply spelling changes

- `Espero que ellos ___ (buscar) una solución.` → `busquen`
- `Es importante que tú ___ (pagar) a tiempo.` → `pagues`
- `Quiero que nosotros ___ (empezar) ahora.` → `empecemos`

### Translation drill

- “I want them to leave early.” → `Quiero que salgan temprano.`
- “It’s possible that she knows the answer.” → `Es posible que ella sepa la respuesta.`
- “We hope that you (plural) come tomorrow.” → `Esperamos que vengáis mañana.`

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Espero que ellos ___ la verdad.' (decir)"
options:
  - "digan"
  - "dicen"
  - "dijeran"
answer: "digan"
feedback:
  correct: "Exacto—usa el presente de subjuntivo 'digan'."
  incorrect: "Recuerda que 'decir' tiene raíz irregular 'dig-' en el subjuntivo." 
```

```exercise
type: fill_in_blank
prompt: "\"Es necesario que nosotros ___ (dormir) más.\" Usa el subjuntivo correcto."
answer: "durmamos"
feedback:
  correct: "Bien—'dormir' cambia o→u en nosotros/vosotros: durmamos." 
  incorrect: "En verbos -ir el cambio se reduce a 'u' en nosotros/vosotros." 
```

```exercise
type: short_answer
prompt: "Escribe dos ejemplos en subjuntivo: uno con un verbo irregular (ser/estar) y otro con un verbo con cambio ortográfico (-car/-gar/-zar)." 
expected_keys:

  - form like busque/pague/empiece
feedback:
  correct: "Excelente—mostraste cómo se aplican las dos irregularidades." 
  incorrect: "Incluye un verbo como 'sea/esté' y otro con 'qu/gu/c' en el subjuntivo." 
```
