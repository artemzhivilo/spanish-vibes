# Lesson Text

Numbers show up in every conversation about schedules, homework, and everyday life. This lesson introduces the core counting pattern so you can talk about quantities with `hay`, ages with `tener`, and descriptions with `ser`.

## Tens and pattern (30–100)

After twenty, numbers combine with `y` (and) plus the unit. The twenties merge into one word, while the rest stay separate.

| Tens | Example | Notes |
| --- | --- | --- |
| 21 = veintiuno | `Hay veintiún estudiantes.` (drop final `o` before masculine noun) |
| 22 = veintidós | accent on last syllable |
| 23 = veintitrés | |
| 24 = veinticuatro | |
| 30 = treinta | `treinta y cinco` |
| 40 = cuarenta | `cuarenta y dos` |
| 50 = cincuenta | `cincuenta y ocho` |
| 60 = sesenta | `sesenta y nueve` |
| 70 = setenta | `setenta y uno` |
| 80 = ochenta | `ochenta y tres` |
| 90 = noventa | `noventa y nueve` |
| 100 = cien / ciento | `cien libros`; `ciento diez tareas` |

**Tip:** Use `ciento` when adding more digits (101–199). For exact 100 say `cien`.

## Using numbers in real sentences

- With `hay`: `Hay treinta estudiantes en la clase.`
- With `ser`: `La biblioteca es grande. Tiene cuarenta mesas.`
- With `tener`: `Tengo veintidós años.`
- With `preguntas`: `¿Cuántos ejercicios hay en la tarea? Hay dieciséis.`

## Fractions of time and money

Even with basic numbers you can express schedules and prices.

- `Son las tres y diez.` (covered more in the time lesson)
- `El café cuesta dos euros.`
- `El examen dura cuarenta minutos.`

## Guided Practice

### Fill in the numbers

1. `15` → `quince`
2. `27` → `veintisiete`
3. `34` → `treinta y cuatro`
4. `58` → `cincuenta y ocho`
5. `99` → `noventa y nueve`
6. `100` → `cien`

### Choose the right form

- `Hay ___ (21) estudiantes.` → `veintiún`
- `Tenemos ___ (30) libros.` → `treinta`
- `La reunión es en el aula ___ (8).` → `ocho`

### Small translations

- “There are forty chairs.” → `Hay cuarenta sillas.`
- “We have sixty minutes.” → `Tenemos sesenta minutos.`
- “The library has one hundred books.” → `La biblioteca tiene cien libros.`

```exercise
feedback:
  correct: "Exact spelling—use 'treinta y ...' with the conjunction y."
  incorrect: "Spell the tens + y + unit as separate words."
```
