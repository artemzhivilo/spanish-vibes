# Lesson Text

Spanish uses **two past tenses**: preterite for completed events, imperfect for background. Mixing them correctly brings stories to life.

## Contrast cheat sheet

| Preterite | Imperfect |
| --- | --- |
| Completed event | Ongoing action |
| Specific time frame | Habitual/repeated |
| Sequence of actions | Description/background |
| Interrupting action | Action being interrupted |
| Sudden change | Mental/physical state |

### Example

- `Estudiaba (imperfecto) cuando sonó (pretérito) el teléfono.`
- `Hacía frío y llovía (imperfecto), pero terminamos el proyecto (pretérito).`

## Decision flow

1. Does the clause describe **what happened** (start/end known)? → Preterite.
2. Does it describe **what was happening** or **what things were like**? → Imperfect.
3. Are you listing actions in a chain? → Preterite.
4. Are you giving age, time, weather, emotion? → Imperfect.

## Guided Practice

### Identify tense

Decide whether each verb should be preterite (P) or imperfect (I).

1. `Mientras ellos ___ (caminar), yo ___ (ver) la noticia.` → `caminaban` (I), `vi` (P)
2. `Ayer nosotros ___ (terminar) el informe.` → `terminamos` (P)
3. `De niños ___ (ir) al lago cada verano.` → `íbamos` (I)
4. `En 2010 ___ (mudarnos) a Madrid.` → `nos mudamos` (P)

### Convert sentences

- `Siempre comimos en la cafetería.` → `Siempre comíamos en la cafetería.` (habitual → imperfect)
- `El año pasado vivíamos en Madrid.` → `El año pasado vivimos en Madrid.` (single completed year → preterite)
- `Mientras estudiábamos, entró la profesora.` (mixed)

### Translation drill

- “It was five o’clock and it was raining.” → `Eran las cinco y llovía.`
- “We finished the task yesterday.” → `Terminamos la tarea ayer.`
- “While I was cooking, my brother arrived.” → `Mientras cocinaba, mi hermano llegó.`

## Mini diálogo

**Elena**: Cuando vivíamos en Sevilla, íbamos al teatro todos los viernes.

**Mateo**: ¡Qué bien! El mes pasado fuimos una vez y nos encantó.

**Elena**: Ayer llovía, pero salimos igual y vimos a unos amigos.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct option: 'Cuando era niño, ___ al parque todos los días.'"
options:
  - "iba"
  - "fui"
  - "iré"
answer: "iba"
feedback:
  correct: "Exacto—acción habitual en la niñez → imperfecto." 
  incorrect: "Para hábitos, usa 'iba'." 
```

```exercise
type: fill_in_blank
prompt: "\"Anoche ___ (hacer) frío y nosotros ___ (quedarse) en casa.\" Usa el tiempo correcto."
answer: "hacía / nos quedamos"
feedback:
  correct: "Bien—'hacía' describe clima (imperfecto) y 'nos quedamos' señala acción puntual (pretérito)." 
  incorrect: "Clima = imperfecto; acción completada = pretérito." 
```

```exercise
type: short_answer
prompt: "Escribe dos oraciones que combinen pretérito e imperfecto para mostrar una interrupción." 
expected_keys:
  - imperfect clause (estaba, caminaba, etc.)
  - preterite clause (llegó, sonó, etc.)
feedback:
  correct: "Excelente—marcaste acción en progreso + interrupción." 
  incorrect: "Incluye estructuras como 'Mientras... (imperfecto), ... (pretérito)'." 
```
