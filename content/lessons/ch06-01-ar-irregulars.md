# Lesson Text

Some high-frequency `-ar` verbs change their stem vowel in most forms. The endings stay regular, but the vowel inside the stem shifts when stressed. This lesson introduces the two patterns you encounter first: **e → ie** and **o → ue**.

## Pattern overview

Stem changes occur in all forms **except** nosotros/vosotros. Think of the conjugation chart shaped like a boot—the inside of the boot (yo, tú, él/ella/usted, ellos/ellas/ustedes) changes, the outside (nosotros/vosotros) does not.

### e → ie verbs (`pensar`, `cerrar`, `empezar`)

| Pronoun | Pensar (to think / plan) |
| --- | --- |
| yo | pienso |
| tú | piensas |
| él / ella / usted | piensa |
| nosotros / nosotras | pensamos |
| vosotros / vosotras | pensáis |
| ellos / ellas / ustedes | piensan |

Other verbs in this group: `cerrar` (to close), `empezar` (to begin), `recomendar` (to recommend).

### o → ue verbs (`mostrar`, `recordar`, `almorzar`)

| Pronoun | Mostrar (to show) |
| --- | --- |
| yo | muestro |
| tú | muestras |
| él / ella / usted | muestra |
| nosotros / nosotras | mostramos |
| vosotros / vosotras | mostráis |
| ellos / ellas / ustedes | muestran |

Other verbs in this group: `recordar` (to remember), `almorzar` (to have lunch), `contar` (to count/tell), `jugar` (unique: `u → ue`, `juego`).

## Using vocabulary you know

- `Cerramos la biblioteca a las ocho, pero Ana la cierra a las siete.`
- `Los estudiantes piensan en el proyecto el lunes.`
- `¿Muestras tus notas a la profesora?`
- `Recordamos el aniversario cada año.`

Combine with `a` from the previous lesson:

- `Invito a mis amigos cuando juego al baloncesto los domingos.`

## Pronunciation tip

Keep the new diphthong strong: *pi-EN-so*, *jue-go*. The unstressed forms (`pensamos`, `jugamos`) keep the original vowel.

## Guided Practice

### Conjugation choices

Select the correct form.

1. `Nosotros ___ (pensar) en el examen.` → `pensamos`
2. `¿Tú ___ (cerrar) la puerta a las seis?` → `cierras`
3. `Ellos ___ (mostrar) sus proyectos.` → `muestran`
4. `Yo ___ (jugar) al tenis los viernes.` → `juego`
5. `Ana y Luis ___ (recordar) la fecha.` → `recuerdan`

### Complete the sentence

- `La biblioteca ___ (cerrar) a las nueve los lunes.` → `cierra`
- `¿A qué hora ___ (empezar) la tutoría?` → `empieza`
- `En verano ___ (almorzar) en el parque.` → `almorzamos`

### Translation drill

- “They think about the trip every day.” → `Piensan en el viaje todos los días.`
- “Do you (formal) close the classroom at night?” → `¿Cierra usted el aula por la noche?`
- “We don’t play on Sundays.” → `No jugamos los domingos.`

## Mini diálogo

**Profesor**: ¿A qué hora empieza la clase hoy?

**Estudiante**: Empieza a las nueve, pero la profesora juega al tenis antes.

**Profesor**: Entonces cerramos la puerta a las nueve en punto.

**Estudiante**: Perfecto, mostramos el proyecto después.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Yo ___ el libro en la tarde.' (cerrar)"
options:
  - "cierro"
  - "cerro"
  - "cierras"
answer: "cierro"
feedback:
  correct: "Bien—yo keeps the stem change: cierro."
  incorrect: "Remember the yo form uses the changed stem + regular ending."
```

```exercise
type: fill_in_blank
prompt: "\"¿A qué hora ___ (empezar) ustedes la reunión?\" Fill in the plural form."
answer: "empiezan"
feedback:
  correct: "Exacto—ustedes takes the stem change plus -an."
  incorrect: "Use the e→ie stem: empiezan."
```

```exercise
type: translation
prompt: "Translate: 'We remember the date every year.'"
answer: "Recordamos la fecha cada año."
feedback:
  correct: "Muy bien—nosotros forms keep the original stem."
  incorrect: "Nosotros has no stem change: recordamos."
```
