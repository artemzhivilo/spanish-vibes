# Lesson Text

The simple future (`futuro simple`) lets you talk about what **will** happen. Unlike the present or preterite, you attach the endings directly to the infinitive—no stem changes for regular verbs.

## Regular endings (attach to infinitive)

| Pronoun | Ending | Example `hablar` | Example `comer` | Example `vivir` |
| --- | --- | --- | --- | --- |
| yo | -é | hablaré | comeré | viviré |
| tú | -ás | hablarás | comerás | vivirás |
| él / ella / usted | -á | hablará | comerá | vivirá |
| nosotros / nosotras | -emos | hablaremos | comeremos | viviremos |
| vosotros / vosotras | -éis | hablaréis | comeréis | viviréis |
| ellos / ellas / ustedes | -án | hablarán | comerán | vivirán |

**Stress** the ending syllable (`habla-RÉ`). All forms include an accent mark except `nosotros`.

## Common irregular stems

The endings stay the same; only the stem changes.

| Infinitive | Stem | Future example |
| --- | --- | --- |
| tener | tendr- | tendré |
| salir | saldr- | saldrás |
| venir | vendr- | vendremos |
| poder | podr- | podrán |
| poner | pondr- | pondremos |
| decir | dir- | dirá |
| hacer | har- | haremos |
| querer | querr- | querrán |
| saber | sabr- | sabré |
| haber | habr- | habrá (there will be) |

## When to use the simple future

- **Plans or intentions**: `Mañana visitaré el museo.`
- **Predictions/certainty**: `Habrá lluvia esta noche.`
- **Promises/threats**: `Te llamaré más tarde.`
- **Probabilities about the present** (informal): `¿Dónde estará Ana?` (I wonder where Ana is.)

## Guided Practice

### Conjugation drill

1. `Nosotros ___ (visitar) a la profesora mañana.` → `visitaremos`
2. `Yo ___ (salir) de la oficina a las seis.` → `saldré`
3. `¿Tú ___ (poder) venir a la reunión?` → `podrás`
4. `Ana y Luis ___ (hacer) un proyecto juntos.` → `harán`
5. `La biblioteca ___ (abrir) temprano el lunes.` → `abrirá`

### Fill in the chart

Complete with the correct irregular stem + ending.

- `Nosotros ___ (tener)` → `tendremos`
- `Ellos ___ (decir)` → `dirán`
- `Yo ___ (querer)` → `querré`

### Translation practice

- “We will study in the library this afternoon.” → `Estudiaremos en la biblioteca esta tarde.`
- “Will you call me tomorrow?” → `¿Me llamarás mañana?`
- “There will be a meeting on Friday.” → `Habrá una reunión el viernes.`

## Mini diálogo

**Lucía**: ¿Qué harás este fin de semana?

**Raúl**: Visitaré a mi familia y luego estudiaré para el examen.

**Lucía**: Perfecto, nosotros tendremos tutoría el lunes.

**Raúl**: Entonces terminaré el proyecto antes del domingo.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Ellos ___ al museo el sábado.' (ir)"
options:
  - "irán"
  - "van"
  - "fueron"
answer: "irán"
feedback:
  correct: "Exacto—usa el futuro simple 'irán'."
  incorrect: "Necesitamos el futuro: irán."
```

```exercise
type: fill_in_blank
prompt: "\"Yo ___ (hacer) la presentación mañana.\" Usa el futuro."
answer: "haré"
feedback:
  correct: "Bien—'hacer' usa el radical irregular 'har-'."
  incorrect: "Recuerda que 'hacer' pierde la 'c': haré."
```

```exercise
type: short_answer
prompt: "Responde: ¿Qué estudiarás el próximo mes? Usa 'Estudiaré...'"
expected_keys:
  - "Estudiaré"
feedback:
  correct: "Excelente—mantén la estructura con el futuro."
  incorrect: "Empieza con 'Estudiaré...' para practicar el tiempo futuro."
```
