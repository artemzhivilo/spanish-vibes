# Lesson Text

Building on previous si-clause patterns, this lesson combines conditional structures to narrate complex hypotheticals.

## 1. Review of core patterns

| Type | Structure | Example |
| --- | --- | --- |
| Present/future unreal | `Si + imperfecto subj.` → `condicional simple` | `Si tuviera tiempo, viajaría.` |
| Past unreal | `Si + pluscuamperf. subj.` → `condicional perfecto` | `Si lo hubiéramos sabido, habríamos venido.` |

## 2. Mixed timeline patterns

| Meaning | Structure | Example |
| --- | --- | --- |
| Past condition → Present result | `Si + pluscuamperf. subj.` → `condicional simple` | `Si hubieras estudiado, estarías preparado ahora.` |
| Present condition → Past result | `Si + imperfecto subj.` → `condicional perfecto` | `Si no fueras tan distraído, no habrías perdido el informe.` |
| Past condition → Imperative/present command | `Si + pluscuamperf. subj.` → `imperativo` | `Si ya lo hubieran terminado, avísenme.` |

## 3. Inverted order and emphasis

- `Estarías más tranquilo si hubieras resuelto el problema a tiempo.`
- `Habrías terminado a tiempo si no perdieras el enfoque.`

Use commas when `si` clause comes first; omit otherwise.

## 4. Avoid common mistakes

- Never use `*Si tendría*`; the conditional never appears directly after `si`.
- For real possibilities, use present indicative: `Si tengo tiempo, iré.` (real) vs. `Si tuviera tiempo, iría.` (unreal).

## Guided Practice

### A. Identify the pattern

1. `Si hubiéramos salido antes, no estaríamos atrapados ahora.` → Mixed (past condition → present result)
2. `Si fueras más organizado, habrías terminado ayer.` → Mixed (present condition → past result)
3. `Si estudiaras, aprobarías.` → Type 2
4. `Si hubieras estudiado, habrías aprobado.` → Type 3

### B. Complete sentences

1. `Si tú ___ (prestar) más atención, no habrías cometido el error.`
2. `Si nosotros ___ (planear) mejor, ahora ___ (tener) más recursos.`
3. `Si ellos ___ (ser) más puntuales, no estarían esperando todavía.`
4. `Si yo ___ (ahorrar) antes, hoy ___ (poder) invertir.`

### C. Transform statements

1. `No ahorramos; ahora no podemos viajar.` → `Si hubiéramos ahorrado, podríamos viajar ahora.`
2. `No eres paciente; ya te habrías recuperado.` → `Si fueras paciente, ya te habrías recuperado.`
3. `Si llegan temprano, avísenme.` (change to past condition) → `Si hubieran llegado temprano, avísenme.`

### D. Translation drill

- “If they had sent the documents, we would be working on them now.”
- “If you weren’t allergic, you would have eaten the dessert.”
- “If we had known, we wouldn’t have come.”
- “If you studied more, you would be less stressed.”

## Mini diálogo

**Ana**: Si hubiéramos pedido ayuda, no estaríamos tan atrasados.

**Luis**: Sí, y si fuéramos más organizados, ya habríamos entregado el proyecto.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct completion: 'Si él hubiera sido más cuidadoso, no ___ el archivo.' (perder)"
options:
  - "habría perdido"
  - "perdería"
  - "hubiera perdido"
answer: "habría perdido"
feedback:
  correct: "Exacto—resultado irreal en el pasado." 
  incorrect: "Recuerda la estructura: si + pluscuamperfecto subj., condicional perfecto." 
```

```exercise
type: fill_in_blank
prompt: "\"Si nosotros ___ (ahorrar) antes, ahora ___ (poder) invertir más.\" Completa con la combinación adecuada."
answer: "hubiéramos ahorrado / podríamos"
feedback:
  correct: "Bien—condición pasada con resultado presente." 
  incorrect: "Usa 'hubiéramos ahorrado' y 'podríamos'." 
```
