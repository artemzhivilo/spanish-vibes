# Lesson Text

You met the preterite in Chapter 8. This lesson tightens your control over regular forms so you can narrate past events fluently. We focus on spotting time cues, combining verbs in sequence, and avoiding present-tense slips.

## Endings refresher

### -ar verbs (`visitar`)

| Pronoun | Ending | Example |
| --- | --- | --- |
| yo | -é | visité |
| tú | -aste | visitaste |
| él / ella / usted | -ó | visitó |
| nosotros / nosotras | -amos | visitamos |
| vosotros / vosotras | -asteis | visitasteis |
| ellos / ellas / ustedes | -aron | visitaron |

### -er / -ir verbs (`aprender`, `vivir`)

| Pronoun | Ending | Aprender | Vivir |
| --- | --- | --- | --- |
| yo | -í | aprendí | viví |
| tú | -iste | aprendiste | viviste |
| él / ella / usted | -ió | aprendió | vivió |
| nosotros / nosotras | -imos | aprendimos | vivimos |
| vosotros / vosotras | -isteis | aprendisteis | vivisteis |
| ellos / ellas / ustedes | -ieron | aprendieron | vivieron |

## Narrative formulas

**Sequencing words** + preterite = clear storyline.

- `Primero`, `Luego`, `Después`, `Más tarde`, `Finalmente`.
- `Ayer`, `Anoche`, `El lunes pasado`, `Hace dos días`, `Por fin`.

### Example paragraph

> `Ayer llegué temprano al campus, preparé los materiales y empecé la clase a las nueve. Después almorzamos en la cafetería y finalmente terminamos el proyecto a las cinco.`

## Common pitfalls (and fixes)

- Forgetting accents on `-é`, `-ó`, `-í`, `-ió` → read aloud emphasizing final syllables.
- Mixing present for nosotros forms (`hablamos`) → rely on time marker (`ayer`) to confirm preterite meaning.
- Overusing imperfect cues like `siempre` → swap for finite markers when narrating a single event (`aquella vez`, `de repente`).

## Guided Practice

### Fill the blanks with preterite endings

1. `Los estudiantes ___ (preparar) una presentación.` → `prepararon`
2. `Nosotros ___ (completar) las actividades en clase.` → `completamos`
3. `¿Tú ___ (compartir) tus apuntes?` → `compartiste`
4. `Ella ___ (abrir) el laboratorio a las ocho.` → `abrió`
5. `Yo ___ (asistir) a la tutoría el jueves.` → `asistí`

### Sequence building

Provide logical connectors + verbs in preterite.

- `___ llegué, ___ saludé a la profesora, ___ nos sentamos.` → `Primero llegué, luego saludé a la profesora, después nos sentamos.`
- `___ revisamos las respuestas y ___ enviamos el proyecto.` → `Después revisamos las respuestas y finalmente enviamos el proyecto.`

### Translation drill

- “Yesterday we finished the exercise quickly.” → `Ayer terminamos el ejercicio rápido.`
- “Did you (plural) call the director last night?” → `¿Llamaron al director anoche?`
- “She studied in the library for three hours.” → `Estudió en la biblioteca durante tres horas.`

## Mini diálogo

**Carla**: ¿Qué hiciste el fin de semana?

**Andrés**: Visité a mis padres, cocinamos juntos y vimos una película.

**Carla**: ¡Qué bien! Yo trabajé el sábado y descansé el domingo.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct completion: 'Anoche ellos ___ la tarea.' (terminar)"
options:
  - "terminaron"
  - "terminan"
  - "terminaban"
answer: "terminaron"
feedback:
  correct: "Exacto—usa el pretérito para una acción completada anoche."
  incorrect: "Busca el marcador temporal 'anoche'; pide pretérito."
```

```exercise

  incorrect: "Coloca el acento en la í: leíste."
```
