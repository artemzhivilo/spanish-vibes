# Lesson Text

Spanish attaches the simple future endings directly to the infinitive to express certainty, plans, or **probability in the present**.

## 1. Endings

Add to the full infinitive:

| Person | Ending | Example (`hablar`) |
| --- | --- | --- |
| yo | -é | hablaré |
| tú | -ás | hablarás |
| él/ella/usted | -á | hablará |
| nosotros/as | -emos | hablaremos |
| vosotros/as | -éis | hablaréis |
| ellos/ellas/ustedes | -án | hablarán |

Accents appear on all forms except nosotros.

## 2. Irregular stems (endings stay the same)

| Verb | Stem | Example |
| --- | --- | --- |
| decir | dir- | diré |
| hacer | har- | harás |
| poner | pondr- | pondremos |
| salir | saldr- | saldrán |
| tener | tendr- | tendrá |
| venir | vendr- | vendremos |
| poder | podr- | podremos |
| querer | querr- | querrás |
| saber | sabr- | sabrán |
| haber | habr- | habrá |
| caber | cabr- | cabremos |

## 3. Uses of the simple future

- **Future plans/intentions**: `Mañana viajaremos a Granada.`
- **Promises/threats**: `Te llamaré esta noche.`
- **Predictions**: `Lloverá esta tarde.`
- **Assumptions/probability about the present**: `¿Dónde estará Ana? (I wonder where Ana is.)`, `Tendrá unos 30 años.`

Use context or adverbs (`probablemente`, `quizá`) to indicate conjecture: `Estarán en casa ahora.`

## Guided Practice

### A. Conjugate to future

1. `nosotros / estudiar` → `estudiaremos`
2. `yo / salir` → `saldré`
3. `ellos / poder` → `podrán`
4. `tú / decir` → `dirás`
5. `usted / querer` → `querrá`

### B. Probability statements

Transform into future tense to express assumption.

1. `Supongo que Mario está en la oficina.` → `Mario estará en la oficina.`
2. `Creo que tienen unos cuarenta años.` → `Tendrán unos cuarenta años.`
3. `Quizá Ana ya sabe la noticia.` → `Ana sabrá la noticia.`

### C. Translation drill

- “We will finish the project tomorrow.”
- “Will you come to the meeting?”
- “It’s 8 p.m.; they are probably at home.”
- “I will call you later.”

## Mini diálogo

**Inés**: ¿Qué harás este fin de semana?

**Diego**: Saldré con mis amigos y luego estudiaré para el examen. ¿Y tú?

**Inés**: Visitaré a mi familia; estarán contentos de verme.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Nosotros ___ el resultado mañana.' (saber)"
options:
  - "sabremos"
  - "sabíamos"
  - "sabremosos"
answer: "sabremos"
feedback:
  correct: "Exacto—futuro con raíz sabr-." 
  incorrect: "Recuerda: saber → sabr- + terminación." 
```

```exercise
type: fill_in_blank
prompt: "\"¿Dónde ___ (estar) la directora ahora?\" Usa el futuro para conjetura."
answer: "estará"
feedback:
  correct: "Bien—futuro expresa probabilidad actual." 
  incorrect: "Expresa 'I wonder where...' con futuro: estará." 
```

```exercise
type: short_answer
prompt: "Escribe seis oraciones en futuro simple: tres para planes reales con marcadores temporales y tres para conjeturas sobre el presente." 
expected_keys:
  - 3 planes (mañana, la próxima semana...)
  - 3 probabilidades (Estará..., Tendrán...)
feedback:
  correct: "Excelente—diferencias claras de uso." 
  incorrect: "Incluye marcadores y contexto de conjetura." 
```
