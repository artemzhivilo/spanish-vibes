# Lesson Text

Spanish uses both the preterite and imperfect to talk about the past. Picking the right one helps listeners understand whether an action was completed, repeated, or simply background information.

## Quick contrast

| Preterite | Imperfect |
| --- | --- |
| Completed action | Habitual, repeated action |
| Specific time frame | No clear beginning/end |
| Sequence of events | Ongoing background |
| Interrupting event | Interrupted action |
| Change of state | Description of state |

### Example

- `Ayer estudié tres horas.` (completed action → preterite)
- `Cuando estudiaba, escuchaba música.` (ongoing background → imperfect)
- `Estudiaba cuando sonó el teléfono.` (ongoing + interruption)

## Common cue words

Preterite: `ayer`, `anoche`, `el año pasado`, `una vez`, `de repente`, `por fin`.

Imperfect: `siempre`, `a menudo`, `cada día`, `de niño`, `mientras`, `generalmente`.

## Choosing step by step

1. **Is the action finished or counted?** → Preterite.
2. **Was it habitual/ongoing?** → Imperfect.
3. **Does it describe age, weather, time, or emotion?** → Imperfect.
4. **Does it interrupt another action?** → Preterite for the interruption, imperfect for the ongoing action.

## Combined examples

- `Era primavera y hacía sol.` (imperfect background)
- `Fuimos al parque y comimos helado.` (preterite sequence)
- `Mientras hablábamos, empezó a llover.` (imperfect + preterite)
- `Cuando yo tenía diez años, vivíamos en Lima.` (imperfect description/habit)

## Guided Practice

### Decide the tense

Identify whether each verb should be in the preterite (P) or imperfect (I).

1. `Cuando era niña, ___ (ir) al museo cada sábado.` → I → `iba`
2. `Ayer ___ (tener) una reunión importante.` → P → `tuve`
3. `Mientras nosotros ___ (estudiar), ellos ___ (llegar).` → `estudiábamos` (I), `llegaron` (P)
4. `Generalmente, mis amigos ___ (cenar) juntos.` → I → `cenaban`

### Convert statements

Rewrite with the correct tense.

1. `Siempre comimos en la cafetería.` → `Siempre comíamos en la cafetería.`
2. `El año pasado vivíamos en Madrid.` → `El año pasado vivimos en Madrid.`
3. `De repente, empezó a llover.` → already correct (preterite)

### Short translations

- “It was five o’clock and it was raining.” → `Eran las cinco y llovía.`
- “We went to the library yesterday.” → `Fuimos a la biblioteca ayer.`
- “While I was studying, my friend called.” → `Mientras estudiaba, mi amigo llamó.`

## Mini diálogo

**Marta**: Cuando era niña, vivía en un pueblo pequeño.

**Diego**: ¿Y visitabas la ciudad a menudo?

**Marta**: No mucho. Pero el año pasado fui a la ciudad y vi a mis amigos de la infancia.

**Diego**: ¡Qué bien! Yo siempre estudiaba allí, pero una vez tomé vacaciones.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct option: 'Ayer ___ mucho calor.'"
options:
  - "hizo"
  - "hacía"
  - "hace"
answer: "hizo"
feedback:
  correct: "Exacto—se refiere a un evento puntual ayer."
  incorrect: "Para una referencia específica ayer, usa el pretérito 'hizo'."
```

```exercise
type: fill_in_blank
prompt: "\"Cuando éramos jóvenes, ___ (jugar) al fútbol todos los días.\" Completa con el imperfecto."
answer: "jugábamos"
feedback:
  correct: "Bien—el hábito en el pasado requiere 'jugábamos'."
  incorrect: "Usa el imperfecto para hábitos: jugábamos."
```
