# Lesson Text

This lesson wraps up the imperative system: special tú forms, irregular formal commands, and collective commands (`nosotros`, `vosotros`).

## Irregular affirmative tú recap

| Verb | Command |
| --- | --- |
| decir | di |
| hacer | haz |
| ir | ve |
| venir | ven |
| salir | sal |
| poner | pon |
| tener | ten |
| ser | sé |

Negative forms use the subjunctive: `no digas`, `no vayas`, `no seas`.

## Irregular formal commands

Because formal commands use the subjunctive, any irregular subjunctive carries over:

| Verb | Usted | Ustedes |
| --- | --- | --- |
| ser | sea | sean |
| estar | esté | estén |
| ir | vaya | vayan |
| saber | sepa | sepan |
| dar | dé | den |
| haber | haya | hayan |

Add pronouns and accents as needed: `Démelo`, `No me lo dé`.

## Nosotros commands

Use the first-person plural present subjunctive to say “let’s …”

- `hablemos`, `comamos`, `escribamos`

For reflexive verbs, drop the final **-s** before attaching **nos**:

- `sentarse` → `sentémonos`
- `irse` → `vámonos` (not *nos vayamos* in affirmation; use `no nos vayamos` for negative)

Negative nosotros commands use `no` + subjunctive: `No hablemos`, `No nos sentemos`.

## Vosotros commands (Spain)

- **Affirmative**: replace the **-r** in the infinitive with **-d**: `hablad`, `comed`, `abrid`.
  - With reflexive **os**, drop the **-d** and attach `-os`: `sentad` → `sentaos`, `vestid` → `vestíos`. Exception: `idos` for `irse`.
- **Negative**: use the present subjunctive vosotros form: `No habléis`, `No comáis`, `No os vayáis`.

## Stem-change patterns summary

- Affirmative tú: mirrors present indicative 3rd singular (keeps stem changes).
- Negative tú / formal / nosotros / vosotros: use subjunctive stems (keep stem changes except nosotros/vosotros for -ar/-er verbs).

## Guided Practice

### Complete the chart

| Verb | Affirmative tú | Negative tú | Usted | Nosotros |
| --- | --- | --- | --- | --- |
| salir | sal | no salgas | salga | salgamos |
| ir | ve | no vayas | vaya | vayamos / vámonos |
| ser | sé | no seas | sea | seamos |

### Create nosotros commands

1. `Trabajar en grupo.` → `Trabajemos en grupo.`
2. `No perder la calma.` → `No perdamos la calma.`
3. `Sentarse aquí.` → `Sentémonos aquí.`
4. `Irse temprano.` → `Vámonos temprano.`

### Vosotros practice

1. `Hablar más bajo.` → `Hablad más bajo.`
2. `No / comer / en clase.` → `No comáis en clase.`
3. `Aplicarse la crema.` → `Aplicad la crema / aplicaos la crema.`

### Translation drill

- “Let’s review the vocabulary.” → `Repasemos el vocabulario.`
- “Don’t let’s forget the tickets.” → `No olvidemos las entradas.`
- “Leave now (vosotros).” → `Salid ahora.`

## Exercises

```exercise
type: mcq
prompt: "Choose the correct nosotros command: '___ al cine esta noche.' (ir)"
options:
  - "Vayamos"
  - "Nos vamos"
  - "Vamos a ir"
answer: "Vayamos"
feedback:
  correct: "Bien—'vayamos' es la forma subjuntiva estándar para mandatos." 
  incorrect: "Los mandatos de nosotros usan el subjuntivo: vayamos." 
```

```exercise
type: fill_in_blank
prompt: "\"No ___ (irse, vosotros) todavía.\" Usa el imperativo negativo."
answer: "os vayáis"
feedback:
  correct: "Exacto—'No os vayáis' usa el subjuntivo vosotros." 
  incorrect: "Acuérdate de añadir el pronombre antes del verbo." 
```

```exercise
type: short_answer
prompt: "Escribe una serie de instrucciones para tu equipo (al menos tres mandatos): uno con 'nosotros', uno con 'ustedes' y uno con 'tú'."
expected_keys:
  - mandato nosotros
  - mandato ustedes
  - mandato tú
feedback:
  correct: "Excelente—combinaste distintos niveles de cortesía." 
  incorrect: "Incluye ejemplos como 'Empecemos...', 'Entreguen...', 'Revisa...'." 
```
