# Lesson Text

Beyond the simple future, Spanish uses **future progressive** and **future perfect** to describe ongoing or completed future actions and to speculate about the present/past.

## 1. Future progressive (futuro continuo)

`estar (futuro) + gerundio`

- `Estaré trabajando a las ocho.`
- `¿Qué estarás haciendo mañana a esta hora?`
- Expresses ongoing action at a specific future moment or polite speculation.

| Person | Form |
| --- | --- |
| yo | estaré + gerundio |
| tú | estarás + gerundio |
| él/ella/usted | estará + gerundio |
| nosotros/as | estaremos + gerundio |
| vosotros/as | estaréis + gerundio |
| ellos/ellas/ustedes | estarán + gerundio |

## 2. Future perfect (futuro perfecto)

`haber (futuro) + participio`

- `Habré terminado el informe para las cinco.`
- `Para 2030, habremos reducido las emisiones.`
- Conveys completion before a future deadline or assumption about the past: `Habrá salido ya` (“He must have left already”).

## 3. Probability nuances

| Tense | Use | Example |
| --- | --- | --- |
| Simple future | present-time probability | `Estará en su oficina.` |
| Future progressive | current activity speculation | `Estarán almorzando ahora.` |
| Future perfect | past probability | `Habrá olvidado la cita.` |

## Guided Practice

### A. Build future progressive

1. `Nosotros / estudiar / a las diez.` → `Estaremos estudiando a las diez.`
2. `¿Tú / trabajar / mañana en la mañana?` → `¿Estarás trabajando mañana en la mañana?`
3. `Ellos / viajar / durante el invierno.` → `Estarán viajando durante el invierno.`

### B. Build future perfect

1. `Para julio, yo / ahorrar / suficiente dinero.` → `Habré ahorrado suficiente dinero.`
2. `Para cuando llegues, nosotros / preparar / la cena.` → `Habremos preparado la cena.`
3. `¿Qué crees? Él / olvidar / el documento.` → `Habrá olvidado el documento.`

### C. Identify tense and meaning

Mark FP (future progressive) or FPerf (future perfect):

1. `Estaré conduciendo durante la tarde.` → FP (action in progress)
2. `Habrán terminado las obras para el viernes.` → FPerf (completed before deadline)
3. `¿Dónde estará?` → Simple future (probability now)
4. `Habrá perdido el tren.` → FPerf (probability about past)

### D. Translation drill

- “We’ll be reviewing the budget at that time.”
- “By next year, they will have implemented the new system.”
- “He must have gone out.”
- “I wonder what the team is doing right now.”

## Mini diálogo

**Coordinadora**: Mañana a las diez estaremos presentando los resultados.

**Analista**: Perfecto. Para entonces habremos revisado todos los datos.

**Coordinadora**: ¿Y Juan?

**Analista**: Estará viajando, pero habrá enviado el informe antes de salir.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct completion: 'Para las ocho ustedes ___ (terminar) el trabajo.'"
options:
  - "habrán terminado"
  - "terminarán"
  - "estarán terminando"
answer: "habrán terminado"
feedback:
  correct: "Exacto—acción completada antes de un punto futuro." 
  incorrect: "'Para las ocho' pide futuro perfecto." 
```

```exercise	type: fill_in_blank
prompt: "A esta hora mañana, nosotros ___ (estar + viajar) a Madrid." Usa el futuro progresivo.
answer: "estaremos viajando"
feedback:
  correct: "Bien—futuro de estar + gerundio." 
  incorrect: "Recuerda formar 'estaremos + viajando'." 
```
