# Lesson Text

This workshop pulls together everything you have learned about the **present subjunctive**. Focus on recognizing triggers, forming verbs correctly, and correcting common mistakes.

## Quick checklists

### Formation recap

1. Start with the present **yo** form.
2. Drop the **-o** ending.
3. Add subjunctive endings:
   - **-ar** → **-e, -es, -e, -emos, -éis, -en**
   - **-er/-ir** → **-a, -as, -a, -amos, -áis, -an**
4. Apply spelling changes (c→qu, g→gu, z→c) and irregular stems (teng-, hag-, conozc-, etc.).

### Trigger checklist

- **Wishes/Influence**: querer que, necesitar que, insistir en que
- **Emotions**: alegrarse de que, temer que, sorprender que
- **Impersonal expressions**: es necesario que, es posible que, es una lástima que
- **Requests**: pedir que, recomendar que, sugerir que
- **Doubt/Denial**: dudar que, no creer que, negar que
- **Ojalá**, **Quizás**, **Tal vez** (with uncertainty)

## Error-spotting guide

| Mistake | Fix |
| --- | --- |
| *Es importante que estudias* | `Es importante que estudies.` |
| *No creo que él viene* | `No creo que él venga.` |
| *Queremos que ellos trabajan* | `Queremos que ellos trabajen.` |
| *Es verdad que ellos lleguen* (certainty) | Use indicative: `Es verdad que ellos llegan.` |

## Guided Practice

### Correct the sentences

Rewrite with the correct subjunctive form.

1. `Es probable que tú terminas temprano.`
2. `Mis padres quieren que yo tengo buenas notas.`
3. `No creemos que ellos entienden la situación.`
4. `Ojalá que ustedes encuentran trabajo pronto.`

### Complete with the subjunctive

- `Es necesario que tú ___ (revisar) el contrato.`
- `Me alegro de que ellos ___ (venir).`
- `Buscamos un lugar que ___ (tener) buena conexión.`
- `Quizás Ana ___ (estar) enferma.`

### Translation drill

- “I hope that you have a great day.”
- “It’s possible that they will call later.”
- “We insist that you submit the form today.”
- “They doubt that the plan will work.”

## Mini diálogo

**Laura**: Es importante que prepares la presentación esta noche.

**Marcos**: Sí, y quiero que revises los datos para que no cometamos errores.

**Laura**: Ojalá que el equipo nos apoye.

**Marcos**: No dudo que lo harán si explicamos el proyecto bien.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'No es cierto que ellos ___ la verdad.' (decir)"
options:
  - "digan"
  - "dicen"
  - "dirán"
answer: "digan"
feedback:
  correct: "Exacto—la negación de certeza requiere subjuntivo." 
  incorrect: "Usa 'digan' porque 'no es cierto que' expresa duda." 
```

```exercise
type: fill_in_blank
prompt: "\"Es mejor que nosotros ___ (salir) temprano.\" Usa el subjuntivo."
answer: "salgamos"
feedback:
  correct: "Bien—'salir' toma raíz irregular: salgamos." 
  incorrect: "Forma el subjuntivo desde 'salgo' → 'salgamos'." 
```

```exercise
type: short_answer
prompt: "Escribe cuatro oraciones que incluyan distintos disparadores del subjuntivo (deseo, emoción, duda, expresión impersonal)."
expected_keys:
  - verbos apropiados
  - subjuntivo presente
feedback:
  correct: "Excelente—cubriste varios disparadores." 
  incorrect: "Incluye ejemplos como 'Espero que...', 'Me molesta que...', 'Dudo que...', 'Es necesario que...'." 
```
