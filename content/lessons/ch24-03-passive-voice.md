# Lesson Text

Spanish offers two main passive constructions—**ser + participle** and **se + verb**—plus stylistic active alternatives. Understanding when to use each keeps your writing natural and clear.

## 1. Ser + participle passive

Structure: `sujeto paciente + ser (tiempo) + participio (+ por + agente)`

- `El informe fue aprobado por el comité.`
- `Las invitaciones serán enviadas mañana.`
- `La decisión ha sido comunicada.`

Use when:

- You want to name the agent (`por el comité`).
- The action is formal or official (reports, announcements).
- You need control over tense (preterite, future, perfect, etc.).

## 2. Se passive (pasiva refleja)

Structure: `se + verbo (3ª persona) + complemento`

- `Se aprobó el informe.`
- `Se enviarán las invitaciones mañana.`
- `Se buscan voluntarios.`

Characteristics:

- Agent is unknown/irrelevant.
- Verb agrees with the patient: singular noun → singular verb (`Se vendió la casa`); plural noun → plural verb (`Se vendieron las casas`).
- Frequently used in instructions, news, signage.

## 3. Active voice alternative

Sometimes active voice feels more direct:

- Passive: `La reunión fue cancelada por la directora.`
- Active: `La directora canceló la reunión.`

Choose active when the agent is important or when clarity/trust demands explicit responsibility.

## 4. Pronoun placement

- With ser + participle + pronouns: `El informe le fue enviado` (rare, formal).
- Se passive avoids direct object pronouns; for personal a + se, switch to active voice: `Se entrevistó a los candidatos.`

## 5. Common pitfalls

| Issue | Incorrect | Correct |
| --- | --- | --- |
| Agreement | *Se publicaron el artículo* | `Se publicó el artículo.` /
`Se publicaron los artículos.` |
| Personal a | *Se vio los estudiantes* | `Se vio a los estudiantes` (keep `a`). |
| Agent misuse | *Se aprobó por el comité* | Use ser-passive: `Fue aprobado por el comité.` |

## Guided Practice

### A. Transform active ↔ passive

1. `El equipo completó el proyecto.` → `El proyecto fue completado por el equipo.`
2. `Los estudiantes entregarán las tareas mañana.` → `Las tareas serán entregadas mañana.`
3. `Se publicó el artículo en marzo.` → Active: `Publicaron el artículo en marzo.`
4. `Se vendieron todas las entradas.` → Active: `Vendieron todas las entradas.`

### B. Choose ser-passive or se-passive

1. (formal announcement) `___ (ser/se) inaugurará el edificio por el alcalde.` → `Será inaugurado por el alcalde.`
2. (signage) `___ (ser/se) prohíbe fumar.` → `Se prohíbe fumar.`
3. (news) `___ (ser/se) detuvieron a dos sospechosos.` → `Se detuvo a dos sospechosos.`

### C. Fill in blanks

1. `La investigación ___ (ser + publicar) en una revista científica.`
2. `___ (se + buscar) voluntarios para el evento.`
3. `___ (ser + celebrar) la ceremonia en el auditorio principal.`
4. `___ (se + vender) productos orgánicos.`

### D. Translation drill

- “The report was written by Marta.”
- “Volunteers are needed.”
- “The winners will be announced tomorrow.”
- “They interviewed the candidates yesterday.” (convert to passive)

## Mini diálogo

**Locutor**: Se anuncian los resultados del concurso.

**Reportera**: Sí, fueron publicados hace unos minutos por el jurado.

**Locutor**: ¿Se entrevistó a todos los participantes?

**Reportera**: Claro, los organizadores realizaron las entrevistas durante la mañana.

## Exercises

```exercise
type: mcq
prompt: "Choose the best option: '___ (Ser/Se) venden libros usados aquí.'"
options:
  - "Se"
  - "Son"
  - "Ser"
answer: "Se"
feedback:
  correct: "Exacto—la pasiva refleja usa 'se'." 
  incorrect: "Emplea 'se venden' para carteles y anuncios." 
```

```exercise
type: fill_in_blank
prompt: "\"El acuerdo ___ (ser + firmar) por ambas partes.\" Usa la voz pasiva."
answer: "fue firmado"
feedback:
  correct: "Bien—acción puntual → 'fue firmado'." 
  incorrect: "Necesitas 'ser' en pretérito + participio." 
```
