# Lesson Text

The **imperfect** sets the background: descriptions, habits, ongoing actions. Its forms are mostly regular, so once you learn the endings, you can focus on meaning.

## Regular endings

### -ar verbs (`trabajar`)

| Pronoun | Form |
| --- | --- |
| yo | trabajaba |
| tú | trabajabas |
| él / ella / usted | trabajaba |
| nosotros / nosotras | trabajábamos |
| vosotros / vosotras | trabajabais |
| ellos / ellas / ustedes | trabajaban |

### -er / -ir verbs (`comer`, `vivir`)

| Pronoun | Comer | Vivir |
| --- | --- | --- |
| yo | comía | vivía |
| tú | comías | vivías |
| él / ella / usted | comía | vivía |
| nosotros / nosotras | comíamos | vivíamos |
| vosotros / vosotras | comíais | vivíais |
| ellos / ellas / ustedes | comían | vivían |

**All -er/-ir endings carry an accent on the í.**

## Only three irregular imperfect verbs

| Infinitive | Forms |
| --- | --- |
| ser | era, eras, era, éramos, erais, eran |
| ir | iba, ibas, iba, íbamos, ibais, iban |
| ver | veía, veías, veía, veíamos, veíais, veían |

## Pronunciation tips

- Stress the antepenultimate syllable: `tra-ba-JA-ba`, `com-Í-a-mos`.
- Keep vowel sounds pure; no diphthongs appear in endings.

## Guided Practice

### Conjugation drill

1. `Yo ___ (estudiar)` → `estudiaba`
2. `Nosotros ___ (vivir)` → `vivíamos`
3. `¿Tú ___ (leer)`? → `leías`
4. `Ellas ___ (trabajar)` → `trabajaban`
5. `Uds. ___ (ser)` → `eran`

### Identify irregulars

- `Cuando era niño, yo ___ (ir) a la escuela en bicicleta.` → `iba`
- `Siempre ___ (ver) dibujos animados.` → `veía`
- `Mi abuela ___ (ser) muy cariñosa.` → `era`

### Translation practice

- “We used to live near the coast.” → `Vivíamos cerca de la costa.`
- “They were friends.” → `Ellos eran amigos.`
- “I was going to the market every Saturday.” → `Iba al mercado todos los sábados.`

## Mini diálogo

**Rosa**: ¿Cómo eras de niña?

**Pablo**: Era tímido, pero iba al parque todos los días y veía caricaturas por la tarde.

**Rosa**: Yo trabajaba con mis padres en la tienda los fines de semana.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Cuando éramos pequeños, ___ mucho tiempo al aire libre.' (pasar)"
options:
  - "pasábamos"
  - "pasamos"
  - "pasábamosos"
answer: "pasábamos"
feedback:
  correct: "Exacto—usa el imperfecto con hábito infantil." 
  incorrect: "Recuerda la tilde en -ábamos." 
```

```exercise
type: fill_in_blank
prompt: "\"Yo ___ (ver) películas los domingos.\" Usa el imperfecto."
answer: "veía"
feedback:
  correct: "Bien—ver toma 'veía' con acento." 
  incorrect: "Añade el acento en la í: veía." 
```
