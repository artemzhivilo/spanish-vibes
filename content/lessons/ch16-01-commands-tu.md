# Lesson Text

Tú commands are the backbone of informal Spanish. Master both affirmative and negative forms to give clear instructions to friends, classmates, or family members.

## Affirmative tú commands

Affirmative tú commands use the **third-person singular** (él/ella/usted) form of the present indicative.

| Infinitive | 3rd person singular | Tú command |
| --- | --- | --- |
| hablar | habla | ¡Habla! |
| comer | come | ¡Come! |
| escribir | escribe | ¡Escribe! |

### Irregular affirmative commands

Some verbs have unique forms:

| Verb | Command |
| --- | --- |
| decir | di |
| hacer | haz |
| ir | ve |
| poner | pon |
| salir | sal |
| ser | sé |
| tener | ten |
| venir | ven |

## Negative tú commands

Use the **present subjunctive** second-person singular: start with the yo form, drop **-o**, add **-es** for -ar verbs and **-as** for -er/-ir verbs.

| Infinitive | Negative command |
| --- | --- |
| hablar | no hables |
| comer | no comas |
| escribir | no escribas |

Irregular yo forms and stem-changing verbs keep their irregularities:

- `no digas`, `no tengas`, `no vengas`, `no hagas`
- `no pienses`, `no vuelvas`, `no pidas`

## Pronoun placement

- **Affirmative**: attach pronouns to the end (add accent if needed): `dime`, `háblame`, `levántate`, `ponlo`.
- **Negative**: place pronouns before the verb: `no me digas`, `no te levantes`, `no lo pongas`.

### Accent rule

When attaching one pronoun to an affirmative command, add an accent to preserve stress: `háblame`, `dímelo`. Count back three syllables and accent the original stressed vowel.

## Common tú command phrases

| Spanish | English |
| --- | --- |
| ¡Cuéntame! | Tell me! |
| ¡No te preocupes! | Don’t worry! |
| ¡Hazlo ahora! | Do it now! |
| ¡Ven aquí! | Come here! |
| ¡Sal de ahí! | Get out of there! |
| ¡Ten paciencia! | Be patient! |
| ¡Sé amable! | Be kind! |

## Guided Practice

### Affirmative or negative?

Fill each blank with the correct form.

1. `___ (venir) pronto.` → `Ven`
2. `___ (no / hablar) durante la prueba.` → `No hables`
3. `___ (poner) la mesa.` → `Pon`
4. `___ (no / salir) tan tarde.` → `No salgas`
5. `___ (escuchar) la explicación.` → `Escucha`

### Pronoun practice

- `Decirme la verdad.` → `Dímela.`
- `No / despertarse / tarde.` → `No te despiertes tarde.`
- `Escribir / la / ahora.` → `Escríbela ahora.`
- `No / enviarlo / todavía.` → `No lo envíes todavía.`

### Translation drill

- “Call me after class.” → `Llámame después de clase.`
- “Don’t forget your homework.” → `No olvides tu tarea.`
- “Be careful!” → `¡Ten cuidado!`

## Mini diálogo

**Ana**: ¿Me ayudas con este informe?

**Luis**: Claro, envíamelo ahora y revísalo antes de compartirlo.

**Ana**: Gracias. No me digas el resultado hasta mañana.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct tú command: '___ la puerta, por favor.' (cerrar)"
options:
  - "Cierra"
  - "Cierres"
  - "Cerraste"
answer: "Cierra"
feedback:
  correct: "Exacto—usa la forma él/ella del presente para el imperativo afirmativo." 
  incorrect: "Recuerda: cerrAr → cierra." 
```

```exercise
type: fill_in_blank
prompt: "\"No ___ (decir) mentiras.\" Usa el imperativo negativo."
answer: "digas"
feedback:
  correct: "Bien—'no digas' viene del subjuntivo (yo digo → no digas)." 
  incorrect: "Forma el subjuntivo: yo digo → digas." 
```

```exercise
type: short_answer
prompt: "Escribe dos mandatos afirmativos y dos negativos dirigidos a un amigo, usando pronombres en al menos uno de ellos." 
expected_keys:
  - al menos 2 comandos afirmativos
  - al menos 2 negativos
  - pronombre adjunto o previo
feedback:
  correct: "Excelente—mostraste control de los comandos tú." 
  incorrect: "Incluye ejemplos como 'Cómpralo' / 'No lo compres'." 
```
