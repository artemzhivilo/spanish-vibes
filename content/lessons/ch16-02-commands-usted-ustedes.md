# Lesson Text

Formal commands address people you respect or don’t know well. They use the **present subjunctive** forms for **usted** and **ustedes**.

## Formation steps

1. Start with the **yo** form of the present indicative.
2. Drop the **-o** ending.
3. Add the subjunctive endings:
   - **-ar verbs**: `-e` (usted), `-en` (ustedes)
   - **-er/-ir verbs**: `-a` (usted), `-an` (ustedes)

### Examples

| Infinitive | Usted | Ustedes |
| --- | --- | --- |
| hablar | hable | hablen |
| comer | coma | coman |
| abrir | abra | abran |
| escribir | escriba | escriban |

Stem changes and yo irregularities carry over:

- `tener` → tenga / tengan
- `salir` → salga / salgan
- `pensar` → piense / piensen
- `volver` → vuelva / vuelvan
- `pedir` → pida / pidan

## Spelling changes

Keep the same orthographic shifts as in the subjunctive:

- `buscar` → busque / busquen
- `llegar` → llegue / lleguen
- `empezar` → empiece / empiecen
- `escoger` → escoja / escojan
- `seguir` → siga / sigan

## Pronoun placement

- **Affirmative**: attach pronouns to the command; add accent if needed: `Hágalo`, `Siéntense`, `Llévese esto`.
- **Negative**: place pronouns before the verb: `No lo haga`, `No se sienten`, `No me diga`.

### Accent rule

When adding one pronoun to a command with more than one syllable, place an accent to maintain stress: `Dígale`, `Escúchenme`, `Siéntense`.

## Useful formal commands

| Spanish | English |
| --- | --- |
| Pase, por favor. | Please come in. |
| Tome asiento. | Take a seat. |
| Lea el informe. | Read the report. |
| No se preocupe. | Don’t worry. |
| Prepárenlo para mañana. | Prepare it for tomorrow. |
| Hable más despacio. | Speak more slowly. |
| Dígame la verdad. | Tell me the truth. |

## Guided Practice

### Form the command

1. `Señora, ___ (abrir) la ventana.` → `abra`
2. `Profesores, ___ (revisar) los exámenes.` → `revisen`
3. `Doctor, no ___ (fumar).` → `fume`
4. `Clientes, no ___ (entrar) sin cita.` → `entren`
5. `Profesor, ___ (explicar) el ejercicio otra vez.` → `explique`

### Pronouns

- `Entregarlo / ustedes` → `Entréguenlo.`
- `No / preocuparse / usted` → `No se preocupe.`
- `Decirme / usted` → `Dígame.`
- `No / repetirlo / ustedes` → `No lo repitan.`

### Translation drill

- “Bring the documents, please.” → `Traiga los documentos, por favor.`
- “Don’t forget to call me.” → `No se olvide de llamarme.`
- “Sit down and listen.” → `Siéntese y escuche.`

## Mini diálogo

**Recepcionista**: Pase, por favor. Tome asiento.

**Visitante**: Gracias. Dígame cómo puedo ayudar.

**Recepcionista**: Espere un momento y firme aquí, por favor.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct formal command: 'Señores, ___ silencio.' (guardar)"
options:
  - "guarden"
  - "guarda"
  - "guardan"
answer: "guarden"
feedback:
  correct: "Exacto—para ustedes use la forma -en." 
  incorrect: "Recuerda: yo guardo → guarden." 
```

```exercise
type: fill_in_blank
prompt: "\"No ___ (sentarse, ustedes) aquí.\" Usa el imperativo negativo."
answer: "se sienten"
feedback:
  correct: "Bien—los pronombres van antes del verbo en negativos." 
  incorrect: "Es 'No se sienten', no 'No sienten'." 
```

```exercise
type: short_answer
prompt: "Escribe tres mandatos formales (dos afirmativos y uno negativo) incluyendo al menos un pronombre." 
expected_keys:
  - dos comandos afirmativos
  - un comando negativo
  - al menos un pronombre adjunto o previo
feedback:
  correct: "Excelente—demostraste cortesía formal." 
  incorrect: "Incluye ejemplos como 'Tráigamelo' o 'No lo repita'." 
```
