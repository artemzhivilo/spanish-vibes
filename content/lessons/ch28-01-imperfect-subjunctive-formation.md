# Lesson Text

The **imperfect subjunctive** expresses past desires, doubts, or hypotheticals. Derive it from the third-person plural preterite.

## 1. Formation steps

1. Conjugate the verb in **ellos/ellas/ustedes preterite**.
2. Remove the ending **-ron**.
3. Add imperfect subjunctive endings.

### -ra variants (most common)

| Person | Ending | Example (`hablar`) |
| --- | --- | --- |
| yo | -ra | hablara |
| tú | -ras | hablaras |
| él/ella/usted | -ra | hablara |
| nosotros/as | -ramos (+ accent on preceding vowel) | habláramos |
| vosotros/as | -rais | hablarais |
| ellos/ellas/ustedes | -ran | hablaran |

*Accents in nosotros/as fall on the vowel before -ramos: `habláramos`, `comiéramos`, `viviéramos`.*

### -se variants (formal/literary)

| Person | Ending | Example |
| --- | --- | --- |
| yo | -se | hablase |
| tú | -ses | hablases |
| él/ella/usted | -se | hablase |
| nosotros/as | -semos | hablásemos |
| vosotros/as | -seis | hablaseis |
| ellos/ellas/ustedes | -sen | hablasen |

Both sets are interchangeable; -ra forms are more common in speech.

## 2. Irregular stems

Irregularities come from the preterite base:

| Verb (preterite ellos) | Stem | Imperfect subjunctive (yo) |
| --- | --- | --- |
| estar (estuvieron) | estuv- | estuviera / estuviese |
| tener (tuvieron) | tuv- | tuviera |
| decir (dijeron) | dij- | dijera |
| traer (trajeron) | traj- | trajera |
| ir/ser (fueron) | fu- | fuera |
| poder (pudieron) | pud- | pudiera |
| poner (pusieron) | pus- | pusiera |
| saber (supieron) | sup- | supiera |
| querer (quisieron) | quis- | quisiera |
| venir (vinieron) | vin- | viniera |
| dar (dieron) | die- | diera |
| haber (hubieron) | hub- | hubiera |

## Guided Practice

### A. Convert preterite to imperfect subjunctive (-ra)

1. `ellos cantaron` → `cantaran`
2. `ellos tuvieron` → `tuvieran`
3. `ellos pidieron` → `pidieran`
4. `ellos fueron` → `fueran`
5. `ellos pusieron` → `pusieran`

### B. Practice -se forms

1. `Si ellos (ir)` → `si ellos fuesen`
2. `Cuando tú (tener)` → `cuando tú tuvieses`
3. `No pensé que nosotros (hacer)` → `no pensé que nosotros hiciésemos`

### C. Fill-in-the-blank

- `Si yo ___ (poder), ayudaría más.`
- `Queríamos que ustedes ___ (venir).`
- `No creían que ella ___ (saber) la respuesta.`
- `Buscábamos a alguien que ___ (hablar) japonés.`

### D. Translation drill

- “If I had more time, I would study another language.”
- “They wanted me to stay.”
- “We didn’t think that you (plural) were telling the truth.”
- “I was looking for a place that was quiet.”

## Mini diálogo

**Lucía**: Si tuviéramos presupuesto, lanzaríamos el proyecto ahora.

**Raúl**: Sí, pero nadie creía que fuera posible este año.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Si nosotros ___ más tiempo, viajaríamos.' (tener)"
options:
  - "tuviéramos"
  - "tengamos"
  - "teníamos"
answer: "tuviéramos"
feedback:
  correct: "Exacto—cláusula hipotética usa imperfecto de subjuntivo." 
  incorrect: "Necesitas 'tuviéramos'." 
```

```exercise
type: fill_in_blank
prompt: "\"No pensaban que tú ___ (venir).\" Usa el imperfecto de subjuntivo."
answer: "vinieras"
feedback:
  correct: "Bien—'venir' → 'viniera/vinieses'." 
  incorrect: "Recuerda la raíz vin- del pretérito." 
```

  incorrect: "Incluye verbos como 'pudiera', 'supiera', 'dijera'."
