# Lesson Text

Progressive tenses highlight that an action is **in progress**. Spanish uses a form of **estar** plus a **gerund** (present participle ending in -ando / -iendo).

## Forming the gerund

- **-ar verbs** → stem + **-ando**: `hablar` → `hablando`
- **-er/-ir verbs** → stem + **-iendo**: `comer` → `comiendo`, `vivir` → `viviendo`
- Stem ending in a vowel → **-yendo** to avoid triple vowels: `leer` → `leyendo`, `traer` → `trayendo`
- Irregular -ir stem-changers: `dormir` → `durmiendo`, `pedir` → `pidiendo`, `decir` → `diciendo`

## Principal progressive tenses

| Tense | Form | Meaning |
| --- | --- | --- |
| **Presente progresivo** | `estoy estudiando` | I am studying (right now) |
| **Pretérito progresivo** | `estaba estudiando` (imperfect of estar) | I was studying (background) |
| **Pretérito perfecto progresivo** | `he estado estudiando` | I have been studying |
| **Futuro progresivo** | `estaré estudiando` | I will be studying |
| **Condicional progresivo** | `estaría estudiando` | I would be studying |

The present progressive is most common; other progressive forms are used less frequently but follow the same pattern.

## Usage rules

- Use the progressive for actions happening **right now**: `Estoy leyendo el informe.`
- For general habits, stick to the simple present: `Leo cada noche.`
- To emphasize duration or ongoing nature in the past: `Estábamos trabajando cuando entró el jefe.`
- Avoid progressive for future plans—use `ir a + infinitivo` or the simple future instead.

## Pronoun placement

- Attach pronouns to the gerund: `Estoy bañándome`, `Está leyéndolo`.
- Or place them before estar: `Me estoy bañando`, `Lo está leyendo`.
- Add an accent when attaching pronouns: `estudiándolos`, `levantándose`.

## Guided Practice

### Form gerunds

1. `leer` → `leyendo`
2. `dormir` → `durmiendo`
3. `traer` → `trayendo`
4. `seguir` → `siguiendo`
5. `preferir` → `prefiriendo`

### Conjugate estar + gerund

- `Yo / trabajar (present)` → `Estoy trabajando.`
- `Nosotros / comer (imperfect)` → `Estábamos comiendo.`
- `Ellos / estudiar (future)` → `Estarán estudiando.`
- `Tú / preparar (conditional)` → `Estarías preparando.`

### Translation drill

- “I am writing an email right now.” → `Estoy escribiendo un correo ahora mismo.`
- “They were sleeping when the alarm rang.” → `Estaban durmiendo cuando sonó la alarma.`
- “We have been practicing all day.” → `Hemos estado practicando todo el día.`

## Mini diálogo

**Ana**: ¿Qué estás haciendo?

**Luis**: Estoy revisando el informe. ¿Y tú?

**Ana**: Estoy preparándome para la reunión; he estado leyendo notas toda la mañana.

**Luis**: Perfecto. A las cuatro estaremos presentando los resultados.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct progressive form: 'María ___ en la biblioteca.' (leer)"
options:
  - "está leyendo"
  - "lee"
  - "está leye"
answer: "está leyendo"
feedback:
  correct: "Exacto—usa estar + gerundio." 
  incorrect: "Recuerda formar el gerundio con -yendo: leyendo." 
```

```exercise
type: fill_in_blank
prompt: "\"Nosotros ___ (estar + practicar) para el concierto cuando empezó a llover.\" Usa el imperfecto progresivo."

feedback:
  correct: "Bien—imperfecto de estar + gerundio." 
  incorrect: "Necesitas 'estábamos' + el gerundio 'practicando'." 
```
