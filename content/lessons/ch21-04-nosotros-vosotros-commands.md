# Lesson Text

**Nosotros** and **vosotros** commands let you coordinate group decisions. Mastering them helps you sound natural in team settings and when interacting with groups in Spain.

## 1. Nosotros commands (let’s …)

Use the **present subjunctive** for inclusive suggestions:

- **-ar**: `hablemos`, `trabajemos`
- **-er/-ir**: `comamos`, `escribamos`

### Reflexive verbs

- Drop the final **-s** before attaching **nos**: `levantemos + nos → levantémonos`, `sentemos + nos → sentémonos`.
- With `irse`, the affirmative form becomes **vámonos** (not *váyamonos*). Negative retains `no nos vayamos`.

### Pronoun placement

- Affirmative: attach pronouns (`Démoselo`, `Comprémoslo`, `Sentémonos`). Add accent marks (`-émonos`, `-ámonos`).
- Negative: `No nos sentemos`, `No se lo demos` (pronouns before the verb).

### Usage

| Situation | Command |
| --- | --- |
| Decide together | `Colaboremos en este proyecto.` |
| Encourage team | `No nos rindamos.` |
| Plan | `Reunámonos mañana a las nueve.` |
| Warn | `No nos olvidemos de pagar.` |

## 2. Vosotros commands (Spain informal plural)

### Affirmative

- Replace the infinitive ending **-r** with **-d**: `hablar → hablad`, `comer → comed`, `escribir → escribid`.
- Reflexive verbs drop the **-d** and add **-os**: `sentad → sentaos`, `vestid → vestíos`.
  - Exception: `irse` becomes `idos` (often pronounced `iros`).
- Pronouns attach: `Decídmelo`, `Traedlo`, `Preparadla`.

### Negative

- Use the **present subjunctive vosotros** form: `No habléis`, `No comáis`, `No escribáis`, `No os sentéis`.

### Usage contexts

| Situation | Commands |
| --- | --- |
| Classroom | `Abrid el libro`, `No habléis tan alto` |
| Friends | `Venid temprano`, `No os olvidéis de las entradas` |
| Sports | `Corred más rápido`, `No os paréis` |

## 3. Comparison chart

| Command | Affirmative | Negative |
| --- | --- | --- |
| Nosotros | `Hablemos`, `Vámonos`, `Sentémonos` | `No hablemos`, `No nos vayamos`, `No nos sentemos` |
| Vosotros | `Hablad`, `Sentaos`, `Decídmelo` | `No habléis`, `No os sentéis`, `No me lo digáis` |

## Guided Practice

### A. Create nosotros commands

1. `Trabajar en equipo.` → `Trabajemos en equipo.`
2. `No / perder la oportunidad.` → `No perdamos la oportunidad.`
3. `Irnos ahora.` → `Vámonos ahora.`
4. `No / olvidarse / de llamar.` → `No nos olvidemos de llamar.`
5. `Sentarse aquí.` → `Sentémonos aquí.`

### B. Vosotros commands

1. `Leer el artículo.` → `Leed el artículo.`
2. `No / comer / en el laboratorio.` → `No comáis en el laboratorio.`
3. `Prepararse / rápido.` → `Preparaos rápido.`
4. `No / preocuparse.` → `No os preocupéis.`
5. `Decirme la verdad.` → `Decídmela.`

### C. Fill in the blanks

- `___ (No / irnos) sin pagar la cuenta.`
- `___ (Acordarse, nosotros) de enviar el informe.`
- `___ (Sentarse, vosotros) en estas sillas.`
- `___ (No / levantarse, vosotros) todavía.`

### D. Translation drill

- “Let’s check the agenda again.”
- “Let’s not arrive late.”
- “Come back early (vosotros).”
- “Don’t get nervous (vosotros).”

## Mini diálogo

**Coordinador**: Reunámonos cinco minutos antes del evento.

**Equipo**: Perfecto, no nos vayamos hasta revisar la lista.

**Coordinador**: Vosotros, preparaos para recibir a los invitados y no os olvidéis de sonreír.

```exercise
  - "Vayamos"
  - "Vamos"
  - "Vamos a ir"

  correct: "'Vayamos' es la forma estándar en el subjuntivo."

```
