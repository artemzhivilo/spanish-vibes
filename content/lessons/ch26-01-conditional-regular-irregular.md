# Lesson Text

The **simple conditional** expresses hypothetical actions, polite requests, or future-in-the-past. It attaches the conditional endings to the infinitive, similar to the future.

## 1. Endings

| Person | Ending | Example (`hablar`) |
| --- | --- | --- |
| yo | -ía | hablaría |
| tú | -ías | hablarías |
| él/ella/usted | -ía | hablaría |
| nosotros/as | -íamos | hablaríamos |
| vosotros/as | -íais | hablaríais |
| ellos/ellas/ustedes | -ían | hablarían |

Accents on all forms.

## 2. Irregular stems

Same stems as the future; add conditional endings.

| Verb | Stem | Example |
| --- | --- | --- |
| decir | dir- | diría |
| hacer | har- | haríamos |
| poner | pondr- | pondría |
| salir | saldr- | saldrían |
| tener | tendr- | tendrías |
| venir | vendr- | vendríamos |
| poder | podr- | podría |
| querer | querr- | querrías |
| saber | sabr- | sabrían |
| haber | habr- | habría |
| caber | cabr- | cabrían |

## 3. Core uses

- **Hypothetical situations**: `Con más tiempo, viajaría por el mundo.`
- **Polite requests / suggestions**: `¿Podrías ayudarme?`, `Me gustaría saber tu opinión.`
- **Future in the past**: `Dijeron que terminarían el informe hoy.`
- **Conditional statements** (`si + imperfecto subjuntivo → condicional`): `Si tuviera dinero, compraría una casa.`

## Guided Practice

### A. Conjugate to conditional

1. `nosotros / aprender` → `aprenderíamos`
2. `yo / salir` → `saldría`
3. `ellos / tener` → `tendrían`
4. `tú / decir` → `dirías`
5. `usted / poder` → `podría`

### B. Polite requests

Transform to conditional.

1. `¿Puedes pasarme la sal?` → `¿Podrías pasarme la sal?`
2. `¿Quieres venir mañana?` → `¿Querrías venir mañana?`
3. `¿Me ayudas con esto?` → `¿Me ayudarías con esto?`

### C. Translation drill

- “I would travel more if I had time.”
- “Would you call me later?”
- “They would like to attend the concert.”
- “She said she would send the report today.”

## Mini diálogo

**Ana**: ¿Podrías revisar este documento?

**Luis**: Claro, lo revisaría ahora mismo pero necesitaría más contexto. ¿Me lo explicarías?

**Ana**: Por supuesto. Sería genial tener tus comentarios antes de mañana.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Nosotros ___ contigo, pero estamos ocupados.' (salir)"
options:
  - "saldríamos"
  - "saldremos"
  - "salimos"
answer: "saldríamos"
feedback:
  correct: "Exacto—acción hipotética." 
  incorrect: "Usa condicional para 'we would go out'." 
```

```exercise
type: fill_in_blank
prompt: "\"Si yo tuviera tiempo, ___ (aprender) otro idioma.\" Usa el condicional."
answer: "aprendería"
feedback:
  correct: "Bien—respuesta condicional." 
  incorrect: "Recuerda la estructura 'if + imperfect subjunctive, conditional'." 
```
