# Lesson Text

This lesson deepens your command of **tú imperatives**, expanding on irregular forms, spelling changes, pronoun placement, and real-life directions.

## 1. Formation review

### Affirmative (hazlo)

- Use the **third-person singular** (él/ella/usted) present indicative.
- Keep existing stem changes: `piensa`, `vuelve`, `pide`.
- Apply spelling changes: `sacar → saca`, `llegar → llega`, `cruzar → cruza` (no change necessary in affirmative).

**Irregular affirmative tú forms (memorize):**

| Verb | Command | Mnemonic |
| --- | --- | --- |
| decir | di | Ven Di Sal Haz Ten Ve Pon Sé |
| hacer | haz |  |
| ir | ve |  |
| poner | pon |  |
| salir | sal |  |
| ser | sé |  |
| tener | ten |  |
| venir | ven |  |

### Negative (no hables)

- Formed from the **present subjunctive** (`yo` form → drop `-o` → add endings `-es` for -ar, `-as` for -er/-ir).
- Preserve stem changes and yo irregularities: `no digas`, `no vengas`, `no conduzcas`, `no empieces`.
- Apply spelling changes: `c→qu` (`no saques`), `g→gu` (`no llegues`), `z→c` (`no cruces`).

## 2. Pronoun placement

| Type | Rule | Example |
| --- | --- | --- |
| Affirmative | Attach pronouns; add accent to maintain stress | `Dímelo`, `Háblales`, `Levántate` |
| Negative | Pronouns precede verb | `No me digas`, `No lo compres`, `No te levantes` |

Double pronoun order: **indirect → direct** (me/te/le/se + lo/la/los/las). When `le/les` precedes lo/la/los/las, change to **se**: `Díselo`, `No se lo des`.

## 3. Reflexive commands and accents

- Add accent three syllables back from the end: `Cepíllate`, `Prepárate`, `Cómpramelo`.
- Verbs ending in two syllables + pronoun may not need an extra accent: `Dime`, `Dale`.
- Use `te` for tú commands even when encouraging yourself (informal context): `¡Cuídate!`

## 4. Contextual phrases

| Scenario | Spanish command | English |
| --- | --- | --- |
| Giving directions | `Gira a la derecha`, `Sigue recto`, `Busca la señal` | Turn right, go straight, look for the sign |
| Reminders at home | `Apaga las luces`, `Lava los platos`, `No dejes la puerta abierta` | Turn off the lights, wash the dishes, don’t leave the door open |
| Encouragement | `Confía en ti`, `Sonríe`, `No te rindas` | Trust yourself, smile, don’t give up |
| Warnings | `Ten cuidado`, `No toques eso`, `Sal de ahí` | Be careful, don’t touch that, get out of there |

## Guided Practice

### A. Transform statements into commands

1. Tú siempre dices mentiras. → `Di la verdad.`
2. Tú debes escuchar con atención. → `Escucha con atención.`
3. No deberías salir tan tarde. → `No salgas tan tarde.`
4. Necesitas ponerse protector solar. → `Ponte protector solar.`
5. No debes perder los documentos. → `No pierdas los documentos.`

### B. Pronoun placement

Rewrite with the correct position and accent.

1. `Decirme la verdad.` → `Dímela.`
2. `No / decir / me / mentiras.` → `No me digas mentiras.`
3. `Traer / nos / los boletos.` → `Tráenos los boletos.`
4. `No / llevar / lo / al aeropuerto.` → `No lo lleves al aeropuerto.`
5. `Explicar / me / lo / ahora.` → `Explícamelo ahora.`

### C. Fill-in-the-blank

Complete with the correct tú command.

1. `___ (cruzar) la calle con cuidado.`
2. `___ (no / empezar) sin mí.`
3. `___ (buscar) la dirección en el mapa.`
4. `___ (no / irse) todavía.`
5. `___ (lavarse) las manos antes de cocinar.`

### D. Translation

- “Send me the report today.”
- “Don’t forget to call your grandmother.”
- “Put on a jacket; it’s cold.”
- “Come back before midnight.”

## Exercises

```exercise
type: mcq
prompt: "Choose the correct tú command: '___ el computador ahora mismo.' (apagar)"
options:
  - "Apaga"
  - "Apague"
  - "Apagues"
answer: "Apaga"
feedback:
  correct: "Exacto—usa la forma él/ella del presente." 
  incorrect: "Recuerda: tú afirmativo usa la forma del presente indicativo 3.ª persona." 
```

```exercise
type: fill_in_blank
prompt: "\"No ___ (tocar) mi teléfono.\" Usa el imperativo negativo."
answer: "toques"
feedback:
  correct: "Bien—'tocar' requiere cambio ortográfico: no toques." 
  incorrect: "Convierte el yo form 'toco' → 'toques'." 
```

prompt: "Escribe cinco mandatos para un compañero de piso usando al menos dos pronombres en total."
