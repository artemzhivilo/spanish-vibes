# Lesson Text

Formal commands are essential in professional, academic, and service encounters. This lesson adds contextual expressions, cultural notes, and advanced practice with **usted/ustedes** imperatives.

## 1. Formation refresher

- Use **present subjunctive** endings:
  - **-ar** → `-e` (usted), `-en` (ustedes): `hable`, `hablen`
  - **-er/-ir** → `-a`, `-an`: `coma`, `coman`; `escriba`, `escriban`
- Maintain stem changes, yo irregularities, and spelling changes: `piense`, `vuelva`, `diga`, `salga`, `juegue`, `almuerce`, `conozca`.
- Negative commands simply add `no` before the verb: `No hable`, `No escriban`.

## 2. Courtesy softeners

Spanish often softens imperatives with introductory phrases:

| Phrase | Use |
| --- | --- |
| `Por favor, + command` | `Por favor, llene este formulario.` |
| `¿Podría/Podrían...?` | `¿Podría firmar aquí, por favor?` |
| `Le/Les ruego que...` | `Les ruego que presten atención.` |
| `Sea tan amable de...` | `Sea tan amable de esperarme un momento.` |
| `Tenga la bondad de...` | `Tenga la bondad de enviarnos la información.` |

Using these structures with the subjunctive softens tone while remaining formal: `Le ruego que confirme la asistencia.`

## 3. Pronouns and passive constructions

- Attach pronouns to affirmative commands (`Revíselo`, `Siéntense`, `Hágamelo saber`).
- Place pronouns before negative commands (`No lo revise todavía`, `No se sienten allí`).
- Passive alternatives for softer tone: `Se ruega no fumar`, `Favor de no estacionarse` (common in signage).

## 4. Functional situations

| Context | Command(s) |
| --- | --- |
| Office | `Firme al final`, `Envíen la propuesta el lunes`, `No compartan la contraseña.` |
| Healthcare | `Respire profundo`, `No coma nada después de medianoche`, `Tómese esta medicina.` |
| Hospitality | `Síganos, por favor`, `Tomen asiento`, `No use el ascensor durante la alarma.` |
| Customer service | `Indíquenos su número de reserva`, `Espere en la fila`, `Acompáñeme, por favor.` |

## 5. Cultural notes

- In many regions, formal commands appear in professional emails, official announcements, and signage.
- In business writing, combining a polite request with a thank-you softens the tone: `Le agradecería que confirmara la reunión.`

## Guided Practice

### A. Convert instructions

1. `Ud. / llenar / este formulario.` → `Llene este formulario.`
2. `Uds. / entregar / el reporte mañana.` → `Entreguen el reporte mañana.`
3. `Ud. / no / olvidar / traer su identificación.` → `No olvide traer su identificación.`
4. `Uds. / sentarse / en la primera fila.` → `Siéntense en la primera fila.`
5. `Ud. / llamarnos / si tiene preguntas.` → `Llámenos si tiene preguntas.`

### B. Add courtesy expressions

Rewrite using softer phrasing.

1. `Firme aquí.` → `Por favor, firme aquí.` / `Podría firmar aquí, por favor.`
2. `Confirme la asistencia.` → `Le ruego que confirme la asistencia.`
3. `No use el teléfono.` → `Sea tan amable de no usar el teléfono.`

### C. Choose the correct form

1. `___ (Envíe / Envía / Envié) el documento antes de las cinco, señor García.`
2. `No ___ (llegue / llegas / lleguen) tarde, por favor.`
3. `___ (Confirmen / Confirman / Confirme) su asistencia, señoras.`
4. `No ___ (se preocupen / les preocupen / preocúpense) por el cambio.`

### D. Translation drill

- “Please, take a seat.”
- “Don’t forget to attach the invoice.”
- “Call me when you arrive, sir.”
- “Ladies and gentlemen, fasten your seat belts.”

## Mini diálogo

**Recepcionista**: Tome asiento, por favor, y complete este formulario.

**Paciente**: Claro. ¿Podría guardarme mis pertenencias?

**Recepcionista**: Sí, démelas y no se preocupe, las mantendremos seguras.

```exercise
type: mcq
prompt: "Choose the correct formal command: 'Señoras, ___ en fila.' (formar)"
options:
  - "formen"
  - "forma"
  - "forman"
answer: "formen"
feedback:
  correct: "Exacto—para ustedes use la forma -en." 
  incorrect: "Recuerda: yo formo → formen." 
```

```exercise
type: fill_in_blank
prompt: "\"Sea tan amable de no ___ (hablar) durante la presentación.\" Usa el subjuntivo correcto."
answer: "hable"
feedback:
  correct: "Bien—'no hable' mantiene la cortesía formal." 
  incorrect: "Con expresiones de cortesía, usa el subjuntivo: hable." 
```
