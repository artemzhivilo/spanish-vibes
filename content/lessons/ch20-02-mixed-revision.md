# Lesson Text

This mixed review forces you to **choose between subjunctive, indicative, and infinitive** in realistic situations. Pay attention to subject changes, certainty, and tone.

## Strategy reminders

1. **Two subjects + trigger** → subjunctive (`Queremos que vengas`).
2. **Same subject** → infinitive (`Queremos venir`).
3. **Certainty / belief** → indicative (`Creo que viene`).
4. **Negative doubt** → still doubt (`No creo que venga`).
5. **Expressions of emotion** → subjunctive with subject change (`Me alegra que estés aquí`).

## Reading: identify moods

Read the paragraph and mark each verb as **SUBJ**, **IND**, or **INF**.

> _Nuestra directora insiste en que preparemos el informe esta semana. Cree que es posible entregar todo a tiempo, pero no está segura de que el comité lo apruebe. Queremos presentarlo sin cometer errores, así que preferimos revisar cada detalle antes de enviarlo._

## Listening-style prompts (respond in writing)

Imagine you hear the following instructions. Respond using the appropriate mood.

1. “Es dudoso que el avión salga hoy.” → (¿Qué dices a tus compañeros?)
2. “El profesor exige que entreguemos la tarea.” → (¿Qué haces?)
3. “Creo que Ana ya sabe la noticia.” → (¿Cómo reaccionas?)
4. “Nos alegra saber que vienes.” → (Responde con un plan.)

## Guided Practice

### Fill in with the correct form

1. `No creo que ellos ___ (tener) tiempo.`
2. `Esperamos ___ (terminar) a tiempo.`
3. `Es evidente que tú ___ (trabajar) mucho.`
4. `Me preocupa que ustedes no ___ (comer) bien.`
5. `No es cierto que Laura ___ (estar) enferma.`

### Choose the best option

For each sentence, choose subjunctive (S), indicative (I), or infinitive (INF).

- `Es necesario ___ estudiar / estudies / estudias más.`
- `Estoy seguro de que ellos ___ llegan / lleguen / llegar.`
- `Nos molesta que tú no ___ participas / participes / participar.`
- `Quiero ___ que me llames / llamar / llamas.`
- `Tal vez ellos ___ vienen / vengan / venir.`

### Translation drill

- “It’s true that we have the documents.”
- “We doubt that they will answer today.”
- “They prefer to wait.”
- “I need you to call me when you arrive.”

## Mini diálogo

**Carla**: No estoy segura de que la reunión empiece a tiempo.

**Miguel**: Yo creo que sí, pero prefiero llegar temprano para estar listo.

**Carla**: El jefe insiste en que revisemos todo una vez más.

**Miguel**: Está bien; es mejor terminarlo ahora que lamentarlo después.

## Exercises

```exercise
type: mcq
prompt: "Choose the best completion: 'Me sorprende que tú no ___.' (participar)"
options:
  - "participes"
  - "participas"
  - "participar"
answer: "participes"
feedback:
  correct: "Exacto—'me sorprende que' dispara el subjuntivo." 
  incorrect: "Debes usar subjuntivo tras expresiones de emoción." 
```

```exercise
type: fill_in_blank
prompt: "\"Es obvio que ellos ___ (saber) la respuesta.\" Usa el modo adecuado."
answer: "saben"
feedback:
  correct: "Bien—'es obvio que' indica certeza → indicativo." 
  incorrect: "Con certeza, usa el indicativo: saben." 
```
