# Lesson Text

Great news: the imperfect tense has only **three** irregular verbs. Master these forms and you’ll cover most background narration.

## Conjugation charts

### Ser (to be)

| Pronoun | Form |
| --- | --- |
| yo | era |
| tú | eras |
| él / ella / usted | era |
| nosotros / nosotras | éramos |
| vosotros / vosotras | erais |
| ellos / ellas / ustedes | eran |

### Ir (to go)

| Pronoun | Form |
| --- | --- |
| yo | iba |
| tú | ibas |
| él / ella / usted | iba |
| nosotros / nosotras | íbamos |
| vosotros / vosotras | ibais |
| ellos / ellas / ustedes | iban |

### Ver (to see/watch)

| Pronoun | Form |
| --- | --- |
| yo | veía |
| tú | veías |
| él / ella / usted | veía |
| nosotros / nosotras | veíamos |
| vosotros / vosotras | veíais |
| ellos / ellas / ustedes | veían |

> Notice every form of `ver` keeps the stem `ve-` plus regular `-er` endings with accent.

## Usage examples

- `Cuando era niño, era tímido y veía muchos programas educativos.`
- `Íbamos al museo todos los meses.`
- `Los profesores eran pacientes y nos veían estudiar.`

Combine with regular imperfect forms:

- `Mientras él leía, yo veía la televisión.`
- `Siempre íbamos a la biblioteca porque era tranquila.`

## Guided Practice

### Complete the sentences

1. `Cuando ___ (ser, yo) pequeño, jugaba con mis primos.` → `era`
2. `Nosotros ___ (ir) al parque cada sábado.` → `íbamos`
3. `Ellos ___ (ver) películas los fines de semana.` → `veían`
4. `¿Tú ___ (ser) estudiante en esta escuela?` → `eras`
5. `Mis padres ___ (ir) al trabajo en tren.` → `iban`

### Mix with regular forms

- `Cuando éramos jóvenes, ___ (vivir) en un apartamento pequeño.` → `vivíamos`
- `Cada verano íbamos a la playa y ___ (tomar) el sol.` → `tomábamos`
- `Yo veía la televisión mientras mi hermana ___ (hacer) la tarea.` → `hacía`

### Translation drill

- “We were very patient.” → `Éramos muy pacientes.`
- “They used to go to the library every day.” → `Iban a la biblioteca todos los días.`
- “Did you watch movies on Fridays?” → `¿Veías películas los viernes?`

## Mini diálogo

**Ana**: Cuando era estudiante, siempre iba a la biblioteca después de clase.

**Luis**: Yo veía a mis amigos allí; éramos un grupo grande.

**Ana**: Sí, los profesores eran amables y nos ayudaban mucho.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'De niño, yo ___ muy curioso.' (ser)"
options:
  - "era"
  - "fui"
  - "soy"
answer: "era"
feedback:
  correct: "Exacto—usa 'era' para describir características en el pasado."
  incorrect: "Recuerda: 'era' expresa descripción continua en el pasado."
```

```exercise
type: fill_in_blank
prompt: "\"Nosotros ___ (ver) películas todos los domingos.\" Escribe la forma correcta."
answer: "veíamos"
feedback:
  correct: "Bien—'veíamos' combina el radical 've-' con la terminación -íamos."
  incorrect: "No olvides el acento en 'veíamos'."
```

```exercise
type: short_answer
prompt: "Escribe en español: 'We used to go to the park every summer.'"
expected_keys:
  - "íbamos"
  - "cada verano"
feedback:
  correct: "Excelente—'Íbamos al parque cada verano' usa el imperfecto de ir."
  incorrect: "Incluye 'íbamos' (con acento) y un marcador habitual como 'cada verano'."
```
