# Lesson Text

Two advanced conditional forms help describe hypotheticals more precisely:

1. **Conditional progressive** (`estar + gerundio`) → ongoing hypothetical action.
2. **Conditional perfect** (`haber + participio`) → hypothetical action that could have happened but didn’t.

## 1. Conditional progressive (condicional progresivo)

`estar (condicional) + gerundio`

- `Estaría estudiando ahora si tuviera tiempo.`
- `¿Qué estarías haciendo si no trabajaras aquí?`

Conveys continuous action under certain conditions or polite speculation: `Pensé que estarían esperando en la entrada.`

| Person | Form |
| --- | --- |
| yo | estaría + gerundio |
| tú | estarías + gerundio |
| él/ella/usted | estaría + gerundio |
| nosotros/as | estaríamos + gerundio |
| vosotros/as | estaríais + gerundio |
| ellos/ellas/ustedes | estarían + gerundio |

## 2. Conditional perfect (condicional compuesto)

`haber (condicional) + participio`

- `Habríamos viajado, pero no conseguimos visado.`
- `De haber sabido la respuesta, la habría dicho.`
- `Pensé que habrías terminado ya.`

Used with **si + pluscuamperfecto** to express unrealized past conditions: `Si hubiéramos tenido tiempo, habríamos visitado el museo.`

## 3. Contrasts

| Meaning | Structure | Example |
| --- | --- | --- |
| Ongoing hypothetical now | Conditional progressive | `Estaríamos celebrando si hubiéramos ganado.` |
| Hypothetical past result | Conditional perfect | `Habríamos celebrado si hubiéramos ganado.` |

## Guided Practice

### A. Form conditional progressive

1. `Nosotros / trabajar / desde casa (si la empresa lo permitiera).` → `Estaríamos trabajando desde casa.`
2. `¿Qué / tú / hacer / ahora?` → `¿Qué estarías haciendo ahora?`
3. `Ellos / viajar / por Europa (si no hubiera restricciones).` → `Estarían viajando por Europa.`

### B. Form conditional perfect

1. `Si hubiera tenido tu número, ___ (llamarte).` → `te habría llamado`
2. `De haberlo sabido, ___ (venir) antes.` → `habría venido`
3. `Pensé que ustedes ___ (terminar) el informe.` → `habrían terminado`

### C. Translation drill

- “We would be working late if the deadline were tomorrow.”
- “They would have visited us, but they were ill.”
- “She thought you would have finished by now.”
- “If you had told me, I would have helped you.”

### D. Convert sentences

1. `Estaban celebrando.` (hypothetical) → `Estarían celebrando si hubieran ganado.`
2. `Viajaron, pero no tenían dinero.` → `Habrían viajado si hubieran tenido dinero.`
3. `Estoy descansando.` (conditional) → `Estaría descansando si no tuviera que estudiar.`

## Mini diálogo

**Raúl**: Si no trabajara hoy, estaría descansando en la playa.

**Claudia**: Yo habría ido contigo si hubiera conseguido permiso.

**Raúl**: Bueno, habríamos disfrutado mucho.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'Si hubieran venido antes, ___ con nosotros.' (cenar)"
options:
  - "habrían cenado"
  - "cenarían"
  - "estarían cenando"
answer: "habrían cenado"
feedback:
  correct: "Exacto—condicional perfecto para resultado no realizado." 
  incorrect: "Usa 'haber + participio' con 'si hubieran...'." 
```

```exercise
type: fill_in_blank
prompt: "\"Estaríamos ___ (discutir) el tema si el profesor estuviera presente.\" Usa el condicional progresivo."
answer: "discutiendo"
feedback:
  correct: "Bien—'estaríamos discutiendo'." 
  incorrect: "Añade el gerundio 'discutiendo'." 
```
