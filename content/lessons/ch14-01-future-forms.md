# Lesson Text

Chapter 14 in the reference text recaps the preterite tense and its uses. This lesson pulls the highlights together so you can narrate past events with confidence, using clear sequences and completed actions.

## Regular endings recap

### -ar verbs (`viajar`)

| Pronoun | Form |
| --- | --- |
| yo | viajé |
| tú | viajaste |
| él / ella / usted | viajó |
| nosotros / nosotras | viajamos |
| vosotros / vosotras | viajasteis |
| ellos / ellas / ustedes | viajaron |

### -er / -ir verbs (`aprender`, `vivir`)

| Pronoun | Aprender | Vivir |
| --- | --- | --- |
| yo | aprendí | viví |
| tú | aprendiste | viviste |
| él / ella / usted | aprendió | vivió |
| nosotros / nosotras | aprendimos | vivimos |
| vosotros / vosotras | aprendisteis | vivisteis |
| ellos / ellas / ustedes | aprendieron | vivieron |

## Narrative building blocks

- **Completed actions**: `Ayer cerré la tienda a las ocho.`
- **Series of actions**: `Llegamos, comimos y salimos rápidamente.`
- **Conditions no longer in effect**: `Fue camarero por dos años.`

### Time markers to reinforce completion

| Spanish | English |
| --- | --- |
| ayer | yesterday |
| anoche | last night |
| anteayer | the day before yesterday |
| hace + time | ... ago |
| de repente | suddenly |
| por fin | finally |

Combine markers with verbs to anchor the timeline: `Hace dos días estudié todo el tema.`

## Guided Practice

### Fill in regular endings

1. `Nosotros ___ (cerrar) la biblioteca a las nueve.` → `cerramos`
2. `¿Tú ___ (vivir) en Madrid el año pasado?` → `viviste`
3. `Ella ___ (aprender) rápido.` → `aprendió`
4. `Ellos ___ (viajar) a Perú en enero.` → `viajaron`
5. `Yo ___ (abrir) la ventana.` → `abrí`

### Sequence exercise

Use connectors to write preterite sentences.

- `Primero / entregar la tarea / luego / ir a casa.` → `Primero entregamos la tarea y luego fuimos a casa.`
- `De repente / sonar el teléfono / nosotros / contestar.` → `De repente sonó el teléfono y nosotros contestamos.`

### Translation drill

- “Yesterday I studied for three hours.” → `Ayer estudié durante tres horas.`
- “They arrived, ate dinner, and left quickly.” → `Llegaron, cenaron y se fueron rápidamente.`
- “What time did you (plural) go to bed?” → `¿A qué hora se acostaron?`

## Mini diálogo

**Ana**: ¿Qué hiciste anoche?

**Luis**: Preparé la presentación, envié los correos y cerré la oficina a las diez.

**Ana**: ¡Qué productivo! Nosotros terminamos el informe y celebramos con café.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct completion: 'Anteayer ellos ___ el proyecto.' (terminar)"
options:
  - "terminaron"
  - "terminan"
  - "terminaban"
answer: "terminaron"
feedback:
  correct: "Exacto—'anteayer' señala una acción terminada." 
  incorrect: "El marcador 'anteayer' pide pretérito: terminaron." 
```

```exercise
type: fill_in_blank
prompt: "\"Yo ___ (invitar) a mis amigos a cenar anoche.\" Usa el pretérito."
answer: "invité"
feedback:
  correct: "Bien—la forma de yo termina en -é." 
  incorrect: "Recuerda: para verbos -ar, la forma de yo es -é (invité)." 
```

```exercise
type: short_answer
prompt: "Escribe dos oraciones en pretérito que describan acciones consecutivas usando 'luego' y 'por fin'."
expected_keys:
  - "luego"
  - "por fin"
feedback:
  correct: "Excelente—secuencia clara con marcadores temporales." 
  incorrect: "Incluye 'Luego...' y 'Por fin...' con verbos en pretérito." 
```
