# Lesson Text

Reflexive, direct, and indirect object pronouns follow consistent placement rules. Learn them now so your sentences flow naturally.

## Basic placement

1. **Before a single conjugated verb**: `Me lavo las manos.`, `No te duchas temprano.`
2. **Attached to an infinitive**: `Voy a lavarme las manos.` (or `Me voy a lavar...`)
3. **Attached to a gerund**: `Estamos cepillándonos los dientes.` (or `Nos estamos cepillando...`)
4. **Attached to affirmative commands**: `¡Lávate!`, `Siéntense.`
5. **Before negative commands**: `No te levantes.`, `No se olviden.`

## Combining pronouns

When using direct (+) indirect object pronouns together, order is **indirect → direct** (`me/te/le/nos/os/les` + `lo/la/los/las`). Reflexive pronouns behave like indirect ones and come first.

- `Me lo pongo.` (I put it on myself.)
- `Se la damos.` (We give it to her.)
- `¿Te las cepillas?` (Do you brush them?)

### Le/les + lo/la/los/las → se

Replace `le` or `les` with `se` when followed by a direct object pronoun: `Se lo explico.`

## Accent marks when attaching pronouns

- Infinitives with one pronoun: optional accent for clarity (`escribirte`).
- Gerunds with one pronoun: add accent on the vowel preceding `-ndo` (`cepillándose`).
- Commands: often need accent to maintain stress (`Siéntate`, `Dígame`).

## Practice with routines

- `Voy a acostarme temprano.` / `Me voy a acostar temprano.`
- `Estamos preparándonos para la clase.` / `Nos estamos preparando...`
- `Ponte la chaqueta.` / `No te la pongas.`

## Guided Practice

### Choose the correct placement

1. (infinitive) `Necesito ___ (lavarse) las manos.` → `lavarme` (también válido: `me voy a lavar` si reordenas la oración)
2. (gerund) `Estamos ___ (prepararse) para salir.` → `preparándonos`
3. (command) `___ (sentarse, ustedes) aquí.` → `Siéntense`
4. (negative command) `No ___ (olvidarse, tú) del examen.` → `te olvides`

### Combine pronouns

- `Poner / yo / la chaqueta` → `Me la pongo.`
- `Dar / nosotros / el libro / a Ana` → `Se lo damos.`
- `Traer / ustedes / los documentos / a mí` → `Me los traen.`

### Translation drill

- “Don’t fall asleep!” → `¡No te duermas!`
- “Put it on right now.” → `Póntelo ahora mismo.`
- “We are brushing our teeth.” → `Nos estamos cepillando los dientes.` / `Estamos cepillándonos los dientes.`

## Mini diálogo

**Carla**: ¿Me lo puedes explicar otra vez?

**Javier**: Claro, te lo explico ahora mismo.

**Carla**: Gracias. Después, voy a ponérmelo en mi cuaderno.

**Javier**: Perfecto, no te olvides de revisarlo mañana.

## Exercises

```exercise
type: mcq
prompt: "Choose the correctly placed pronouns: '___ voy a poner la chaqueta.'"
options:
  - "Me la"
  - "La me"
  - "La te"
answer: "Me la"
feedback:
  correct: "Exacto—indirect (me) precede al directo (la)."
  incorrect: "Recuerda el orden: me/te/se/nos/os + lo/la/los/las."
```

```exercise
type: fill_in_blank
prompt: "\"No ___ (acostarse, tú) tarde.\" Escribe el mandato negativo correcto."
answer: "te acuestes"
feedback:
  correct: "Bien—para mandatos negativos, pronombre antes del verbo." 
  incorrect: "Invierte el orden y conjuga en subjuntivo negativo: no te acuestes." 
```

```exercise
type: short_answer
prompt: "Transforma en una sola oración: 'Voy a explicar la tarea a ustedes. La tarea es importante.' Usa pronombres." 
expected_keys:
  - "Se la voy a explicar"
  - "Voy a explicársela"
feedback:
  correct: "Excelente—colocaste los pronombres en el orden correcto." 
  incorrect: "Necesitas 'se' (por les) + 'la' para 'la tarea', antes o unido al infinitivo." 
```
