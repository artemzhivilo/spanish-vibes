# Lesson Text

Regular `-er` verbs were straightforward. Now meet the common irregulars that power daily conversations: stem-changing verbs like `querer`, `volver`, and the must-know `tener` and `hacer`.

## e → ie verbs (`querer`, `entender`)

| Pronoun | Querer (to want) |
| --- | --- |
| yo | quiero |
| tú | quieres |
| él / ella / usted | quiere |
| nosotros / nosotras | queremos |
| vosotros / vosotras | queréis |
| ellos / ellas / ustedes | quieren |

`Entender` (to understand) follows the same pattern: `entiendo`, `entiendes`, etc.

## o → ue verbs (`volver`, `poder`)

| Pronoun | Volver (to return) |
| --- | --- |
| yo | vuelvo |
| tú | vuelves |
| él / ella / usted | vuelve |
| nosotros / nosotras | volvemos |
| vosotros / vosotras | volvéis |
| ellos / ellas / ustedes | vuelven |

`Poder` (to be able to/can) is another essential: `puedo`, `puedes`, `puede`, `podemos`, `podéis`, `pueden`.

## Irregular verbs: `tener` and `hacer`

These verbs combine irregular stems with the **yo-go** pattern (`yo tengo`, `yo hago`).

| Pronoun | Tener (to have) | Hacer (to do/make) |
| --- | --- | --- |
| yo | tengo | hago |
| tú | tienes | haces |
| él / ella / usted | tiene | hace |
| nosotros / nosotras | tenemos | hacemos |
| vosotros / vosotras | tenéis | hacéis |
| ellos / ellas / ustedes | tienen | hacen |

Use `tener` for age, obligations (`tener que + infinitive`), possessions. Use `hacer` for weather (`hace frío`), tasks, and activities.

## Using earlier vocabulary

- `Tengo clase a las nueve y hago la tarea por la tarde.`
- `¿Puedes volver a la biblioteca a las cinco?`
- `Queremos visitar a la profesora el lunes.`
- `Los estudiantes entienden el horario nuevo.`

Combine with expressions from Chapter 4–5:

- `Tenemos una reunión el 15 de mayo.`
- `¿Puedes leer el número en voz alta?`

## Vocabulary

spanish | english | Notes
---|---|---
querer | to want / love | stem e → ie
entender | to understand | stem e → ie, alternate: `comprender`
volver | to return | stem o → ue
poder | to be able to | stem o → ue; often with infinitive
tener | to have | irregular; yo-go + stem change
hacer | to do / make | irregular yo hago
traer | to bring | preview: yo traigo
poner | to put/place | preview: yo pongo

## Guided Practice

### Choose the correct form

1. `Nosotros ___ (querer) estudiar hoy.` → `queremos`
2. `¿Tú ___ (poder) venir a las siete?` → `puedes`
3. `Ellos ___ (volver) tarde.` → `vuelven`
4. `Yo ___ (tener) dos clases los martes.` → `tengo`
5. `Ustedes ___ (hacer) ejercicio en el gimnasio.` → `hacen`

### Complete the dialogue

- `¿A qué hora ___ (tener) la reunión?` → `tienes`
- `La reunión ___ (hacer) falta.` (use expression) → `hace`
- `No ___ (entender, yo) el horario nuevo.` → `entiendo`

### Translation drill

- “Can we visit the museum today?” → `¿Podemos visitar el museo hoy?`
- “She has to study at night.” → `Ella tiene que estudiar por la noche.`
- “I do my homework before class.” → `Hago la tarea antes de la clase.`

## Mini diálogo

**Ana**: ¿Puedes venir a la tutoría?

**Luis**: No puedo hoy, tengo una reunión y hago un proyecto.

**Ana**: Entiendo. Entonces volvemos mañana.

**Luis**: Perfecto, quiero repasar las preguntas contigo.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct form: 'María ___ volver temprano.'"
options:
  - "puede"
  - "pueden"
  - "podemos"
answer: "puede"
feedback:
  correct: "Exacto—María/ella takes 'puede'."
  incorrect: "Remember: only ella/él/usted use 'puede'; save 'pueden' or 'podemos' for plural subjects."
```

```exercise
type: fill_in_blank
prompt: "\"Nosotros ___ (tener) examen mañana.\" Fill in the correct form."
answer: "tenemos"
feedback:
  correct: "Bien—nosotros keeps the stem and adds -emos."
  incorrect: "Use tenemos; no stem change in nosotros."
```

```exercise
type: short_answer
prompt: "Answer in Spanish: ¿Qué haces los fines de semana? Usa 'Hago...'"
expected_keys:
  - "Hago"
feedback:
  correct: "Muy bien—mantén la forma irregular 'hago'."
  incorrect: "Empieza con 'Hago...' para practicar el yo-go."
```
