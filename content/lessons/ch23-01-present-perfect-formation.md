# Lesson Text

The **present perfect** links the past to the present. It answers “What have you done?” using **haber (present)** + **past participle**.

## 1. Haber + participle

| Person | Haber | Example |
| --- | --- | --- |
| yo | he | `He estudiado` |
| tú | has | `Has viajado` |
| él/ella/usted | ha | `Ha comido` |
| nosotros/as | hemos | `Hemos trabajado` |
| vosotros/as | habéis | `Habéis leído` |
| ellos/ellas/ustedes | han | `Han aprendido` |

Past participles: stem + `-ado` (-ar) / `-ido` (-er/-ir).

- hablar → `hablado`
- comer → `comido`
- vivir → `vivido`

## 2. Irregular participles (memorize)

`abierto, cubierto, dicho, descrito, descubierto, escrito, hecho, impreso, muerto, puesto, resuelto, roto, satisfecho, visto, vuelto`

Derived verbs share the same base: `devolver → devuelto`, `componer → compuesto`.

## 3. Pronoun placement and negation

- Pronouns precede haber: `Me he levantado`, `No lo hemos visto`, `¿Te has dado cuenta?`
- Never insert pronouns between haber and the participle.

## 4. Time markers and adverbs

| Adverb | Use |
| --- | --- |
| ya | already | `Ya hemos terminado.` |
| todavía (no) | yet / still not | `Todavía no he comido.` |
| alguna vez / nunca / jamás | ever / never | `¿Has viajado alguna vez?` / `Nunca he ido.` |
| últimamente | lately | `Últimamente han salido mucho.` |
| recientemente | recently | `He leído tu informe recientemente.` |

## Guided Practice

### A. Conjugate to present perfect

1. `Nosotros ___ (aprender)` → `hemos aprendido`
2. `¿Tú ___ (ver) la nueva película?` → `has visto`
3. `Ana ___ (escribir) un artículo.` → `ha escrito`
4. `Yo ___ (abrir) la ventana.` → `he abierto`
5. `Ellos ___ (decir) la verdad.` → `han dicho`

### B. Fill in with ya/todavía

1. `¿___ has terminado el informe?`
2. `No, ___ no lo he terminado.`
3. `Nosotros ___ hemos visitado Madrid dos veces.`
4. `Ellos ___ han llegado; esperan afuera.`

### C. Translation drill

- “I have already sent the email.”
- “We haven’t seen each other in months.”
- “Have you ever lived abroad?”
- “They have just finished the presentation.”

## Mini diálogo

**Laura**: ¿Has revisado los datos?

**Diego**: Sí, ya los he revisado y he corregido los errores.

**Laura**: Perfecto. Nosotros todavía no hemos enviado el informe.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct completion: '¿___ hablado con Marta?'"
options:
  - "Has"
  - "Han"
  - "Haber"
answer: "Has"
feedback:
  correct: "Exacto—segunda persona singular." 
  incorrect: "Recuerda: yo he, tú has, él ha..." 
```

```exercise
type: fill_in_blank
prompt: "\"Ellos ___ (romper) el contrato.\" Usa el participio correcto."
answer: "han roto"
feedback:
  correct: "Bien—participio irregular 'roto'." 
  incorrect: "Memoriza: romper → roto." 
```
