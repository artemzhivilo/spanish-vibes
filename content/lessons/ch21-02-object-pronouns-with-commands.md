# Lesson Text

Pronouns transform commands by clarifying **to whom** and **what**. Learn to attach, combine, and accentuate pronouns with confidence.

## 1. Pronoun hierarchy

Order remains consistent:

`me / te / se / nos / os` **before** `lo / la / los / las`

- Le/les becomes **se** when paired with direct-object pronouns: `Dale el libro → Dáselo`, `No se lo des`.
- Reflexive pronouns (me, te, se, nos, os) follow the same rules.

## 2. Affirmative commands

- Pronouns attach to the end of the command.
- Add an accent to preserve stress (count back three syllables).

| Command | + Direct object | + Indirect + Direct | + Reflexive |
| --- | --- | --- | --- |
| `Escribe` | `Escríbelo` | `Escríbeselo` | `Escríbete` |
| `Haz` | `Hazlo` | `Házselo` | `Hazte` |
| `Lleva` | `Llévalo` | `Llévaselo` | `Llévate` |
| `Digan` | `Díganlo` | `Díganselo` | `Díganse` |

When the resulting form has three syllables or more, add an accent on the original stressed syllable: `enviar → envía → envíamelo`.

## 3. Negative commands

- Pronouns precede the verb.
- No accents needed (the verb retains natural stress).

| Command | Negative with pronoun |
| --- | --- |
| `No lo escribas` | direct object |
| `No se lo des` | indirect + direct |
| `No te levantes` | reflexive |
| `No me lo cuentes` | combinations |

## 4. Reflexive + object combos

- Affirmative: `Ponte el abrigo` → `Póntelo`
- Negative: `No te lo pongas`
- Nosotros commands drop the final -s before **nos**: `Vamos + nos → vámonos`, `Sentemos + nos → sentémonos`, `No nos vayamos`.

## 5. Practical scenarios

| Situation | Command with pronouns | Meaning |
| --- | --- | --- |
| Handing documents | `Entrégalos al profesor` / `Entrégaselos` | Hand them to him/her |
| Offering help | `Dame la caja` / `Dámela` | Give it to me |
| Warning | `No se lo digas a nadie` | Don’t tell it to anyone |
| Teaching | `Explícanos el problema` | Explain the problem to us |

## Guided Practice

### A. Attach pronouns (affirmative)

1. `Traer / me / los resultados.` → `Tráemelos.`
2. `Dar / le / el informe.` → `Dáselo.`
3. `Decir / me / la verdad.` → `Dímela.`
4. `Poner / te / la chaqueta.` → `Póntela.`
5. `Hacer / nos / un favor.` → `Háznoslo.` (often written `Háznoslo`)

### B. Negative placement

1. `No / entregar / me / el paquete.` → `No me entregues el paquete.`
2. `No / mostrar / les / las fotos.` → `No se las muestres.`
3. `No / enviar / nos / la invitación.` → `No nos envíes la invitación.`
4. `No / quitar / te / la mascarilla.` → `No te quites la mascarilla.`

### C. Choose the correct form

Select the correct option.

1. `___ (Dámelo / Dame lo / Dame se lo)` ahora mismo.
2. `No ___ (te la pongas / póntela / ponte la)` todavía.
3. `___ (Entréguenselo / Entréguenseló / Entréguense lo)` al director.
4. `No ___ (se lo digas / dígaselo / dígaselo)` a nadie.

### D. Translation drill

- “Send it to me when you finish.”
- “Don’t tell them the surprise.”
- “Give yourself a break.”
- “Put them on and leave.”

## Mini diálogo

**Supervisor**: Entrégales el informe a los clientes.

**Asistente**: Claro, se lo envío ahora mismo. ¿Necesita algo más?

**Supervisor**: Sí, recuérdales la reunión y no les digas la decisión final todavía.

## Exercises

```exercise
type: mcq
prompt: "Choose the correct placement: '___ el documento.' (Give it to me)"
options:
  - "Dámelo"
  - "Me da lo"
  - "Da me lo"
answer: "Dámelo"
feedback:
  correct: "Exacto—pronombres unidos con acento." 
  incorrect: "Pronombres se unen y la tilde mantiene la sílaba tónica." 
```

```exercise
type: fill_in_blank
prompt: "\"No ___ (decir, tú) nada a mis padres.\" Inserta pronombres si es necesario."
answer: "digas"
feedback:
  correct: "Bien—sin pronombres extra, basta 'No digas nada'." 
  incorrect: "Usa el subjuntivo negativo: no digas." 
```

```exercise
type: short_answer
prompt: "Escribe cuatro comandos con doble pronombre (dos afirmativos, dos negativos)." 
expected_keys:
  - afirmativos con pronombres adjuntos
  - negativos con pronombres antes del verbo
feedback:
  correct: "Excelente—controlas las combinaciones." 
  incorrect: "Asegúrate de ordenar se/me/te antes de lo/la/los/las." 
```
